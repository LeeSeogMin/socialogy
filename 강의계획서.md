# 강의계획서: 사회통계 (Social Statistics)
**AI 시대의 데이터 리터러시와 분석 실무**

---

## 1. 교과목 개요 (Course Overview)
본 과목은 사회학적 질문에 답하기 위한 **양적 분석(Quantitative Analysis)**의 기초를 학습하는 과정입니다. 전통적인 통계 이론을 습득하는 동시에, **GitHub Copilot**을 인공지능 조교로 활용하여 코딩, 디버깅, 결과 해석의 언어화 과정을 훈련합니다. 학생들은 실제 데이터를 활용해 '질문-가설-분석-해석-보고'의 전 과정을 경험하며 데이터 기반의 비판적 사고 능력을 기릅니다.

## 2. 수업 철학 및 AI 활용 정책 (Course Philosophy & AI Policy)

### 2.1. "AI 시대형" 운영 철학
- **도구로서의 AI**: GitHub Copilot은 코딩과 해석의 효율을 높여주는 '조교'입니다. 학생은 더 많은 시간을 '질문'과 '판단'에 집중합니다.
- **비판적 검증**: Copilot의 결과물은 반드시 실제 코드 실행 결과와 이론적 근거를 바탕으로 학생이 최종 판단하고 수정해야 합니다.

### 2.2. 상세 활용 가이드
- **허용 범위**: 개념 설명, 분석 절차 안내, 코드 디버깅, 결과 해석 초안 작성, 보고서 문장 다듬기.
- **금지 사항**: 시험 답안 그대로 제출, 데이터 위조/조작, 타인의 과제 복제, 개인정보/민감정보 입력.
- **증빙 의무**: 모든 과제에는 **(1) 결과 근거(표/그래프)**와 **(2) Copilot 사용 전략 요약(2-3줄)**이 반드시 포함되어야 합니다.

## 3. 학습 목표 (Learning Objectives)
학기 말 학생은 다음을 수행할 수 있어야 합니다:
1. 사회학적 데이터를 정제하고 변수의 **측정 수준(Nominal/Ordinal/Interval/Ratio)**을 정확히 구분한다.
2. 분석 목적에 맞는 통계 기법(요약통계, 분할표, t-검정/ANOVA, 회귀분석)을 **적절히 선택하고 실행**한다.
3. 통계 결과에서 핵심 수치를 찾아내어 사회학적 언어로 **명확하게 기술하고 한계를 제시**한다.
4. AI 도구를 활용해 효율적으로 코딩하되, 결과의 정확성을 **비판적으로 검증**한다.

## 4. 평가 방법 (Evaluation)

| 항목 | 비율 | 세부 내용 |
| :--- | :---: | :--- |
| **중간고사** | 30% | 9주차 실시 (객관식 60%, 주관식 40%) |
| **주간 과제 (1-8주)** | 20% | 수업 내 완결형 과제 (과제 1-8) |
| **개인 리서치 브리프** | 40% | 10-15주 개인 프로젝트 (데이터 분석 보고서 + 발표) |
| **출석 및 참여** | 10% | 출결 및 세션 피어 피드백 |

**개인 리서치 브리프 내부 배점** (40% 중):

| 평가 항목 | 비율 | 세부 |
| :--- | :---: | :--- |
| 분석 타당성 | 25% | 척도·가정에 맞는 기법 선택, 가정 확인, 효과크기 보고 |
| 해석 정확성 | 25% | 비인과 표현, 한계 명시, 선행연구 비교 |
| 재현성 | 15% | 노트북 재실행 성공, 데이터 경로·시드 설정 |
| 발표 전달력 | 15% | 5분 엄수, 표/그림 1개 이상 활용, "So what" 명확 |
| Q&A 대응 | 10% | "모르겠다"의 솔직함 + 근거 기반 답변 |
| 세션 기여 | 10% | 다른 발표자에 대한 구체 피드백 작성 |

- **시험 정책**: 시험 중에는 **GitHub Copilot 및 인터넷 사용이 금지**됩니다.
- **AI 양산 방지**: 개인 리서치 브리프는 매주 노트북 "Restart & Run All" 실행 스크린샷을 증빙으로 요구합니다. 텍스트만 제출한 과제는 0점으로 처리합니다.
- **분량 기준**: 최종 리서치 브리프는 **6-8쪽** (A4, 본문 기준). 10쪽 초과는 AI 양산 신호로 간주하여 감점합니다.
- **과제 피드백**: 제출 다음 주 첫 수업 시간에 루브릭 기반 피드백을 제공합니다.
- **출석**: 결석 1회당 2점, 지각 1회당 1점 차감.
