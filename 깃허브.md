# 1회차 강의자료: GitHub Copilot 설치 및 사용법(수업 운영용)

## 1) 오늘의 목표(90분)
- GitHub 계정/학생 인증(가능한 경우)을 통해 **Copilot 사용 준비**를 완료한다.
- VS Code에서 Copilot을 설치하고, **코드 완성(Autocomplete)**과 **Copilot Chat**을 기본 수준으로 사용할 수 있다.
- Copilot을 “정답 생성기”가 아니라 **코딩 조교**로 쓰는 규칙(프라이버시/윤리/검증)을 이해한다.

## 2) 준비물(학생)
- 노트북(Windows/macOS)
- GitHub 계정(없으면 수업 중 생성)
- VS Code 설치 권장
- (가능하면) 학교 이메일 또는 학생증/재학증명서(학생 인증용)

## 3) 수업 중 규칙(중요)
- Copilot이 만든 코드는 **반드시 실행/검증**하고, 결과(출력/표/그래프/테스트)로 근거를 남긴다.
- 개인/민감정보(이름, 전화번호, 학번, 건강·정치·종교 등) 및 비공개 자료를 Copilot/Chat에 입력하지 않는다.
- 과제 제출물에는 “Copilot을 어떻게 활용했는지(2–3줄)”를 적되, **복붙 제출**이 아니라 **본인 이해/근거**가 핵심이다.

## 4) 설치/활성화(수업 중 체크리스트)
### 4-1. GitHub Student Developer Pack(학생 인증, 가능하면 권장)
학생 혜택으로 Copilot을 무료/할인으로 사용할 수 있는 경우가 있다(학교/정책에 따라 다름).
1) GitHub에 로그인 후 학생 팩 페이지로 이동해 신청한다.
2) 학교 이메일 또는 학생증/재학증명서로 학생 신분을 인증한다.
3) 승인 후 Copilot 설정 화면에서 플랜/권한을 확인한다.

> 주의: 결제 정보 요구/유료 전환 화면이 나오면 **즉시 멈추고** 조교/교수에게 확인한다(계정 상태에 따라 다름).

### 4-2. VS Code 설치
- VS Code가 없다면 설치한다(기본 설치로 충분).

### 4-3. Copilot 확장 설치(필수)
1) VS Code 실행
2) Extensions(확장)에서 `GitHub Copilot`, `GitHub Copilot Chat` 설치
3) Command Palette에서 “Copilot: Sign In” → GitHub 로그인
4) 우측 하단/상단 상태에서 Copilot이 활성화되었는지 확인

### 4-4. 동작 확인(3분)
아래 중 1개라도 확인되면 성공.
- 파일에서 타이핑 시 회색 제안(ghost text)이 나타난다.
- Copilot Chat 창에서 질문을 입력하면 답변이 온다.

## 5) Copilot 기본 사용법(수업에서 쓰는 기능만)
### 5-1. 코드 완성(Autocomplete) 잘 쓰는 법
- “짧은 주석 + 함수 시그니처 + 예시 입력” → 제안 품질이 올라간다.
- 한 번에 크게 받지 말고, **작게 생성 → 실행 → 수정**을 반복한다.

좋은 입력 예:
- 주석: “CSV 불러와 결측치 처리 후, 변수 요약표 만들어 출력”
- 함수 이름/인자: `def summarize_survey(df): ...`

### 5-2. Copilot Chat(질문 템플릿)
수업에서 자주 쓰는 질문 템플릿을 통일한다.
- “내 목표는 ___이고, 제약은 ___이다. Python으로 최소 예제를 만들어줘.”
- “이 에러 메시지의 원인 후보 3개와 확인 절차를 순서대로 알려줘.”
- “이 결과를 ‘인과’처럼 읽히지 않게 사회과학 논문 스타일로 5문장으로 써줘.”
- “내 코드가 재현 가능하도록(입력/출력/seed) 구조를 정리해줘.”

### 5-3. 수정/리팩터링 요청(금지어 포함)
Copilot에게 “바꿔달라”가 아니라 “규칙을 지켜 바꿔달라”라고 요청한다.
- “함수 길이 30줄 이하, 변수명 명확히, I/O 분리, 예외 처리 추가”
- “분석 단계(정제→요약→검정→해석)를 함수로 나눠줘”

## 6) 수업 내 실습 과제(30–40분, 수업 내 완결형)
### 과제 1: ‘Copilot을 코딩 조교로 쓰기’ 미션
1) (10분) 아래 중 하나를 선택해 Copilot 도움으로 코드 초안을 만든다.
   - (A) CSV 로드 → 결측치 요약표 출력
   - (B) 범주형 변수 분포표 + 막대그래프
   - (C) 평균/중앙값/최빈값 요약 + 해석문 3문장
2) (10분) 코드를 실행하고, 에러가 나면 Chat으로 원인/해결 절차를 받아 수정한다.
3) (10분) 결과를 5문장으로 해석하되, 아래 2문장을 반드시 포함한다.
   - “이 결과는 ___를 의미하지만, ___를 의미하지는 않는다.”
   - “해석의 한계는 ___이다.”

### 제출물(수업 종료 전 제출)
- (A) 목표 1문장
- (B) Copilot에게 무엇을 요청했는지 2–3줄(요약)
- (C) 실행 결과(표/그래프/출력 중 1개)
- (D) 해석문 5문장

## 7) 자주 발생하는 문제(빠른 해결)
- Copilot 제안이 안 뜸: 로그인 상태 확인 → VS Code 재시작 → 확장 enable 확인
- Chat이 안 보임: `GitHub Copilot Chat` 확장 설치 여부 확인
- 제안이 엉뚱함: 주석을 더 구체화(입력/출력/제약) + 작은 단위로 나눠 요청
- 코드가 돌아가지만 틀림: “테스트/검증 코드 작성”을 먼저 시키고, 결과 근거를 남긴다

## 8) 다음 시간 예고
- 다음 시간부터는 사회통계 주제(요약통계, 검정, 회귀 등)를 다루며,
  Copilot은 **절차 점검/코드 작성/해석문 다듬기**에 제한적으로 활용한다.
