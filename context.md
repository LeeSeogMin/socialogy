# 사회통계 강의자료 프로젝트 컨텍스트

**"사회통계: 사회조사·통계분석 기초 + 프로젝트, GitHub Copilot 활용"** — 주 2회(회당 90분) 수업 설계, 수업 내 완결형 과제, GitHub Copilot 조교 활용에 맞춘 강의자료 자동 제작 프로젝트이다.

## 현재 상태

### 프로젝트 개요
- **구성**: 14주 × 2회차 = 28회차
- **워크플로우**: 5단계 (Planning → Research & Code → Writing → Optimization → Quality Verification)
- **제작 지침**: `contents.md` 참조
- **프로젝트 지침**: `CLAUDE.md` 참조

### 완료된 작업

- ✅ **제작 지침서** (`contents.md`)
  - 강의계획서 작성 원칙, 주차 구성 원칙, 90분 수업 템플릿
  - 수업 내 완결형 과제 설계 지침, Copilot 활용 방식
  - 회차 설계 체크리스트(10항목)

- ✅ **강의계획서** (`syllabus.md`)
  - 14주 × 2회차 주차별 강의계획
  - 평가방법(중간 30% / 기말 30% / 매주 과제 40%)
  - 팀프로젝트(11–14주, 3인 팀) 운영 계획

- ✅ **1회차 자료** (`깃허브.md`)
  - GitHub Copilot 설치/로그인/기본 사용법
  - 수업 중 규칙(검증, 프라이버시, 제출)
  - 미니 실습 과제(Copilot을 코딩 조교로 쓰기)

### 주차별 핵심 주제

| 주차 | 핵심 질문(Driving Question) | 핵심 분석 기법 |
|---:|---|---|
| 1주 | AI 도구를 통계 학습의 조교로 어떻게 활용할까? | 환경 설정, 미니 실습 |
| 2주 | 이 데이터를 믿을 수 있는가? | 데이터 리터러시, 코딩/결측/변수 정의 |
| 3주 | 데이터의 전체 모습을 어떻게 파악하고 표현할까? | 요약·분포·산포, 해석문 구조 |
| 4주 | 두 범주형 변수가 관련 있는가? | 분할표·카이자승(가정/해석) |
| 5주 | 표본의 패턴이 모집단에도 존재하는가? | 가설검정 프레임, p값/유의수준 |
| 6주 | 집단 간 평균이 '진짜' 다른가? | t-검정/ANOVA(가정/해석) |
| 7주 | 한 변수가 다른 변수를 얼마나 예측하는가? | 단순회귀(인과 오해 방지) |
| 8주 | 중간고사 | 대비 + 시험 |
| 9주 | 여러 변수를 동시에 고려하면 해석이 어떻게 달라지는가? | 중다회귀(통제의 의미/한계) |
| 10주 | 질문부터 해석까지 혼자 완주할 수 있는가? | 미니 완주 프로젝트 |
| 11–14주 | 팀으로 하나의 사회과학적 질문에 답하기 | 3인 팀, 체크포인트 4회 + 쇼케이스 |

### 집필 패턴

- **90분 구조**: 질문/목표(0–10분) → 미니 강의(10–30분) → 실습(30–70분) → 정리+비판적 읽기(70–85분) → Exit ticket(85–90분)
- **과제 형식**: A–E 표준 제출물 (분석 질문, 변수/척도, 근거, 해석문, Copilot 사용 요약)
- **문체**: 격식체 평서문, 개조식 허용, 가독성 중심
- **코드 참조**: `_전체 코드는 practice/W{주차}/code/{파일명}.py 참고_`
- **해석문 템플릿**: (분석목적) → (핵심근거) → (결론) → (주의/한계)
