- **구성**: 14주 × 2회차 = 28회차

- **제작 지침**: `contents.md` 참조
- **프로젝트 지침**: `CLAUDE.md` 참조

### 완료된 작업

- ✅ **제작 지침서** (`contents.md`)
  - 강의계획서 작성 원칙, 주차 구성 원칙, 90분 수업 템플릿
  - 수업 내 완결형 과제 설계 지침, Copilot 활용 방식
  - 회차 설계 체크리스트(10항목)

- ✅ **강의계획서** (`syllabus.md`)
  - 14주 × 2회차 주차별 강의계획
  - 평가방법(중간 30% / 기말 30% / 매주 과제 40%)
  - 팀프로젝트(11–14주, 3인 팀) 운영 계획

- ✅ **1회차 자료** (`깃허브.md`)
  - GitHub Copilot 설치/로그인/기본 사용법
  - 수업 중 규칙(검증, 프라이버시, 제출)
  - 미니 실습 과제(Copilot을 코딩 조교로 쓰기)

### 주차별 핵심 주제

| 주차 | 주제 | 핵심 분석 기법 |
|---:|---|---|
| 1주 | 오리엔테이션 + Copilot | 환경 설정, 미니 실습 |
| 2주 | 데이터 준비 | 코딩/결측/변수 정의, 척도 구분 |
| 3주 | 기술통계/시각화 | 요약·분포·산포, 해석문 구조 |
| 4주 | 범주형 관계 | 분할표·카이자승(가정/해석) |
| 5주 | 가설검정 | p값/유의수준/오류 프레임 |
| 6주 | 평균차 검정 | t-검정/ANOVA(가정/해석) |
| 7주 | 회귀 1 | 단순회귀(계수/적합도/인과 오해 방지) |
| 8주 | 중간고사 | 대비 + 시험 |
| 9주 | 회귀 2 | 중다회귀(통제의 의미/한계) |
| 10주 | 통합 정리 | 분석 흐름 완주, 결과 보고 |
| 11–14주 | 팀프로젝트 | 3인 팀, 체크포인트 4회 + 쇼케이스 |

### 집필 패턴

- **90분 구조**: 질문/목표(0–10분) → 미니 강의(10–30분) → 실습(30–70분) → 정리(70–85분) → Exit ticket(85–90분)
- **과제 형식**: A–E 표준 제출물 (분석 질문, 변수/척도, 근거, 해석문, Copilot 사용 요약)
- **문체**: 격식체 평서문, 개조식 허용, 가독성 중심
- **코드 참조**: `_전체 코드는 practice/W{주차}/code/{파일명}.py 참고_`
- **해석문 템플릿**: (분석목적) → (핵심근거) → (결론) → (주의/한계)
