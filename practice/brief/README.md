# 개인 리서치 브리프 템플릿 (10-15주)

이 폴더는 10-15주 개인 리서치 브리프 작성과 주차별 체크포인트 제출에 사용하는 템플릿을 모은 곳이다.

## 파일 목록

### 최종 브리프 (14-15주 제출)
| 파일 | 용도 |
|---|---|
| `research_brief_template.md` | 최종 리서치 브리프 템플릿 (6-8쪽). 본 파일을 복사하여 사용 |
| `research_brief_template.docx` | 동일한 템플릿의 Word 버전. Word 선호 학생용 |

### 주차별 체크포인트
| 파일 | 제출 주 |
|---|---|
| `cp1_data_question.md` | 10주 — 데이터·연구 질문 확정 |
| `cp2_first_test.md` | 11주 — 결측치 처리 + 1차 검정 |
| `cp3_regression.md` | 12주 — 중다회귀 + 본문 초안 4쪽 |
| `cp4_draft_review.md` | 13주 — 완성 초안 + 교차리뷰 |

### 평가·피드백 양식
| 파일 | 용도 |
|---|---|
| `cross_review_form.md` | 13주 동료 교차리뷰 (리뷰어 → 피리뷰자) |
| `peer_feedback_form.md` | 14-15주 세션 피어 피드백 (세션 내 다른 발표자 전원) |
| `self_reflection_memo.md` | 15주 개인 성찰 메모 (1쪽) |

## 양식 포맷 안내

- **기본 포맷**: Markdown (`.md`) — 버전 관리·협업·변환에 유리
- **보조 포맷**: Word (`.docx`) — 최종 브리프만 제공
- **LaTeX는 제공하지 않음** — 학부 사회학 분석 보고서에 과잉 스펙으로 판단. 필요시 학생 자율 선택

## 제출 방식

- **Markdown 제출**: `.md` 파일 그대로 Google Classroom에 업로드 가능
- **PDF 변환**: `pandoc input.md -o output.pdf`
- **DOCX 변환**: `pandoc input.md -o output.docx`
- **Word 제출**: `research_brief_template.docx`를 복사해 작성 후 제출

## 템플릿 사용 원칙

1. **대괄호 안의 안내문구는 삭제**하고 본인 내용으로 채운다 (예: `[데이터 이름]` → `KGSS 2022`)
2. **섹션 제목은 수정하지 않는다** (루브릭 자동 채점 대비)
3. **분량 가이드를 지킨다** — 최종 브리프 6-8쪽 초과 금지 (AI 양산 의심)
4. **자가 점검 체크리스트**를 채우고 제출한다
