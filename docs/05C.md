## A회차 빠른 진단 정답

**Q.** 200명의 설문에서 남성과 여성의 소득 평균이 다르게 나왔다. 다음 중 올바른 해석은?

**정답**: ② 이 표본에서 남녀 소득 차이가 관찰되었으나, 모집단에서도 그런지는 추가 검정이 필요하다. 표본에서 관찰된 패턴이 모집단에도 존재하는지 확인하려면 추론통계(가설검정)가 필요하다.

---

## B회차 빠른 진단 정답

**Q.** "연구 질문이 '교육수준(고졸/대졸/대학원)에 따라 월 소득이 다른가?'일 때, 적절한 검정은 무엇이며 그 이유는?"

**정답**: ANOVA이다. 독립변수(교육수준)가 명목/서열이며 3수준이고, 종속변수(월 소득)가 비율척도이므로, 세 집단 이상의 평균 차이를 검정하는 ANOVA가 적절하다. t-검정은 2수준일 때만 사용한다.

---

## 과제 1 기대 결과: 가설 설정 + 검정 선택

### 질문 1: "성별에 따라 월 소득에 차이가 있는가?"

| 항목 | 기대 답안 |
|---|---|
| H₀ | 남성과 여성의 모집단 월 소득 평균에 차이가 없다 |
| H₁ | 남성과 여성의 모집단 월 소득 평균에 차이가 있다 |
| 독립변수(척도) | 성별 (명목, 2수준: 남성/여성) |
| 종속변수(척도) | 월 소득 (비율) |
| 선택한 검정 | 독립표본 t-검정 |
| 선택 이유 | 독립변수가 명목 2수준, 종속변수가 비율이므로 가이드표에서 독립표본 t-검정에 해당 |

### 질문 2: "교육수준에 따라 생활만족도가 다른가?"

| 항목 | 기대 답안 |
|---|---|
| H₀ | 교육수준(고졸/대졸/대학원) 집단 간 모집단 생활만족도 평균에 차이가 없다 |
| H₁ | 교육수준 집단 중 적어도 하나의 쌍에서 모집단 생활만족도 평균에 차이가 있다 |
| 독립변수(척도) | 교육수준 (서열, 3수준: 고졸/대졸/대학원) |
| 종속변수(척도) | 생활만족도 (서열/등간, 5점 척도) |
| 선택한 검정 | ANOVA |
| 선택 이유 | 독립변수가 3수준, 종속변수가 등간/비율이므로 가이드표에서 ANOVA에 해당 |

**참고**: 교육수준이 서열척도이므로 ANOVA 적용에 대해 논의의 여지가 있다. 그러나 3수준 이상의 집단 평균 비교라는 점에서 ANOVA를 선택하는 것이 일반적이다. 비모수 대안으로 Kruskal-Wallis 검정을 사용할 수도 있다.

### 질문 3: "연령과 소득 사이에 관계가 있는가?"

| 항목 | 기대 답안 |
|---|---|
| H₀ | 모집단에서 연령과 소득 사이에 (선형) 상관이 없다 (rho = 0) |
| H₁ | 모집단에서 연령과 소득 사이에 (선형) 상관이 있다 (rho ≠ 0) |
| 독립변수(척도) | 연령 (비율) |
| 종속변수(척도) | 소득 (비율) |
| 선택한 검정 | 상관분석 / 단순회귀 |
| 선택 이유 | 두 변수 모두 비율(연속)이므로 가이드표에서 상관/회귀에 해당 |

## 과제 2 기대 결과: Copilot 검정 선택 확인

### 기대되는 Copilot 응답 패턴

**질문 1에 대한 Copilot 확인**:
- Copilot이 "독립표본 t-검정을 사용하라"고 답할 가능성이 높다. 이는 가이드표와 일치하므로 **채택**이 적절하다.
- 채택 근거 예시: "Copilot이 '독립표본 t-검정을 사용하라'고 답했다. 채택한다. 독립변수가 명목 2수준(남성/여성)이고 종속변수가 비율(월 소득)이므로, 두 집단 평균 차이를 검정하는 독립표본 t-검정이 가이드표와 일치한다."

**질문 2에 대한 Copilot 확인**:
- Copilot이 "ANOVA를 사용하라"고 답하거나, "Kruskal-Wallis를 고려하라"고 답할 수 있다.
- ANOVA를 제안한 경우 **채택** 근거 예시: "Copilot이 'ANOVA를 사용하라'고 답했다. 채택한다. 교육수준이 3수준이므로 t-검정이 아닌 ANOVA가 적절하다."
- Kruskal-Wallis를 제안한 경우에도, "생활만족도를 등간으로 처리하면 ANOVA가 적절하지만, 서열로 보면 비모수 검정이 더 보수적인 선택이다"라는 근거로 채택 또는 거절 모두 가능하다.

아래는 각 질문을 선택한 경우의 모범 보고서이다.

### 질문 1 선택 시: 성별과 월 소득

#### (A) 분석 질문

성별에 따라 월 소득에 통계적으로 유의한 차이가 있는가?

#### (B) 사용 변수/척도

- 성별: 명목척도 (남성, 여성)
- 월 소득: 비율척도 (만 원)

#### (C) 결과 근거

- 검정: 독립표본 t-검정
- t(198) = 2.45, p = 0.015
- Cohen's d = 0.35
- 남성 평균: 355만 원, 여성 평균: 310만 원 (차이: 45만 원)

#### (D) 해석문

**(분석목적)** 성별에 따라 월 소득에 차이가 있는지 확인하기 위해 독립표본 t-검정을 실시했다.

**(핵심근거)** t(198) = 2.45, p = 0.015이며, Cohen's d = 0.35이다. 남성 평균은 355만 원, 여성 평균은 310만 원으로, 약 45만 원의 차이가 관찰되었다.

**(결론)** 유의수준 0.05에서 귀무가설(남녀 월 소득 평균에 차이가 없다)을 기각한다. Cohen's d = 0.35로, 작은~중간 수준의 효과 크기이다.

**(주의)** 이 결과는 성별이 소득을 "결정"한다는 의미가 아니다. 직종, 경력, 근무시간, 연령 등 교란변수를 통제하지 않았으며, 이 데이터의 표본 대표성에도 한계가 있다.

#### (E) Copilot 사용 요약

Copilot에게 "성별(명목, 2수준)과 월 소득(비율)일 때 어떤 검정을 써야 하는지"를 물어 독립표본 t-검정이라는 답변을 받았다. 가이드표와 일치하여 채택했다. 해석문 작성 시 Copilot이 "성별이 소득에 영향을 미친다"로 서술한 부분을 "성별에 따라 소득에 차이가 관찰된다"로 직접 수정했다.

---

### 질문 2 선택 시: 교육수준과 생활만족도

#### (A) 분석 질문

교육수준(고졸/대졸/대학원)에 따라 생활만족도에 통계적으로 유의한 차이가 있는가?

#### (B) 사용 변수/척도

- 교육수준: 서열척도 (고졸, 대졸, 대학원)
- 생활만족도: 서열/등간척도 (5점 리커트)

#### (C) 결과 근거

- 검정: ANOVA
- F(2, 197) = 4.82, p = 0.009
- eta^2 = 0.047
- 고졸 평균: 2.85, 대졸 평균: 3.15, 대학원 평균: 3.52

#### (D) 해석문

**(분석목적)** 교육수준에 따라 생활만족도에 차이가 있는지 확인하기 위해 ANOVA를 실시했다.

**(핵심근거)** F(2, 197) = 4.82, p = 0.009이며, eta^2 = 0.047이다. 집단별 평균은 고졸 2.85, 대졸 3.15, 대학원 3.52로, 교육수준이 높은 집단에서 생활만족도 평균이 더 높게 나타났다.

**(결론)** 유의수준 0.05에서 귀무가설(세 집단의 생활만족도 평균에 차이가 없다)을 기각한다. eta^2 = 0.047로, 작은~중간 수준의 효과 크기이다. 다만 어느 집단 쌍에서 차이가 유의한지는 사후검정(예: Tukey HSD)이 필요하다.

**(주의)** 이 결과는 교육이 만족도를 "높인다"는 인과를 의미하지 않는다. 소득, 직업, 연령 등 교란변수가 통제되지 않았으며, 생활만족도를 등간척도로 처리한 가정에도 한계가 있다.

#### (E) Copilot 사용 요약

Copilot에게 "교육수준 3수준, 생활만족도 5점 척도일 때 어떤 검정을 사용해야 하는지"를 물어 ANOVA라는 답변을 받았다. 가이드표와 일치하여 채택했다. 추가로 "eta^2 = 0.047이면 효과가 큰 편인가?"라고 물어 "작은~중간 수준"이라는 답변을 받아 해석문에 반영했다.

---

### 질문 3 선택 시: 연령과 소득

#### (A) 분석 질문

연령과 소득 사이에 통계적으로 유의한 (선형) 관계가 있는가?

#### (B) 사용 변수/척도

- 연령: 비율척도 (세)
- 소득: 비율척도 (만 원)

#### (C) 결과 근거

- 검정: 상관분석 / 단순회귀
- r = 0.42, p < 0.001
- beta_1 = 5.2 (연령 1세 증가 시 소득 5.2만 원 증가 추정)

#### (D) 해석문

**(분석목적)** 연령과 소득 사이에 선형 관계가 있는지 확인하기 위해 상관분석 및 단순회귀를 실시했다.

**(핵심근거)** r = 0.42, p < 0.001이다. 단순회귀에서 beta_1 = 5.2로, 연령이 1세 증가할 때 소득이 평균 약 5.2만 원 증가하는 것으로 추정된다.

**(결론)** 유의수준 0.05에서 귀무가설(연령과 소득 사이에 상관이 없다)을 기각한다. r = 0.42로 중간 수준의 양의 상관이 관찰된다.

**(주의)** 이 결과는 연령이 소득을 "결정"한다는 인과를 의미하지 않는다. 경력, 직종, 교육수준 등 교란변수가 통제되지 않았다. 또한 연령과 소득의 관계가 선형이 아닐 가능성(예: 50대 이후 소득 감소)도 고려해야 한다.

#### (E) Copilot 사용 요약

Copilot에게 "두 변수 모두 비율척도(연속)일 때 어떤 검정을 사용해야 하는지"를 물어 "상관분석 또는 단순회귀"라는 답변을 받았다. 가이드표와 일치하여 채택했다. Copilot이 "연령이 소득을 높인다"로 서술한 부분을 "연령과 소득 사이에 양의 상관이 관찰된다"로 직접 수정했다.

## 흔한 오류 3개 + 확인법

### 오류 1: H₀와 H₁ 뒤바꿈

"성별에 따라 소득에 차이가 있다"를 H₀로, "차이가 없다"를 H₁로 설정하는 경우이다. 가설검정은 **"차이/관계가 없다"에서 출발**한다.

**확인법**: H₀에 "없다", "같다", "독립이다" 같은 표현이 있는지 확인한다. 만약 H₀에 "다르다", "있다"가 들어 있으면 뒤바뀐 것이다.

### 오류 2: 변수 척도와 검정 불일치

독립변수가 3수준인데 t-검정을 선택하거나, 두 변수 모두 연속인데 카이자승 검정을 선택하는 경우이다.

**확인법**: 가이드표에서 자신의 변수 척도 조합을 찾아 해당 행의 검정과 일치하는지 확인한다.

### 오류 3: 해석문에서 인과 표현 사용

"교육이 만족도를 높인다", "성별이 소득에 영향을 미친다" 등의 인과 표현이 D항에 포함되는 경우이다.

**확인법**: 해석문에서 다음 단어를 검색하여 수정한다.
- "영향" -> "연관" 또는 "차이"
- "높인다/낮춘다" -> "높게/낮게 나타난다"
- "결정한다" -> "관계가 관찰된다"
- "때문에" -> "에 따라"

**모범 답안**:
1. **적절한 검정**: ANOVA. 독립변수(연령대)가 명목/서열 4수준이고, 종속변수(운동 시간)가 비율척도이므로, 세 집단 이상의 평균 차이를 검정하는 ANOVA가 적절하다.

**채점 포인트**:
- ANOVA 선택 + 이유(3수준 이상, 종속변수 연속): 1점
- H₀가 "차이가 없다"로 올바르게 설정: 1점
- t-검정을 선택한 경우: 독립변수가 4수준이므로 부적절 (감점)
- 카이자승을 선택한 경우: 종속변수가 연속이므로 부적절 (감점)

## 확장(Plus) 트랙 기대 답안

### 3개 분석의 효과 크기 비교

| 질문 | 검정 | 효과 크기 | 해석 기준 | 수준 |
|---|---|---|---|---|
| 질문 1 (성별 → 소득) | t-검정 | Cohen's d = 0.35 | d: 0.2=작음, 0.5=중간, 0.8=큼 | 작은~중간 |
| 질문 2 (교육 → 만족도) | ANOVA | eta^2 = 0.047 | eta^2: 0.01=작음, 0.06=중간, 0.14=큼 | 작은~중간 |
| 질문 3 (연령 → 소득) | 상관/회귀 | r = 0.42 | r: 0.1=작음, 0.3=중간, 0.5=큼 | 중간~큰 |

**기대 답변**: "3개 분석 중 효과 크기가 가장 큰 것은 질문 3(연령과 소득, r = 0.42)이다. Cohen의 기준에서 r = 0.42는 중간~큰 수준의 효과에 해당하며, 질문 1(d = 0.35, 작은~중간)과 질문 2(eta^2 = 0.047, 작은~중간)보다 크다. 다만 효과 크기 지표가 서로 다르므로 직접 비교에는 주의가 필요하다."

**참고**: 서로 다른 효과 크기 지표(d, eta^2, r)를 직접 비교하는 것은 엄밀하지 않다. 동일한 지표로 변환(예: d를 r로 변환)하면 보다 정확한 비교가 가능하지만, 이 과제에서는 각 지표의 해석 기준에 따른 "수준" 비교로 충분하다.
