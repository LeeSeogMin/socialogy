# W11. 팀으로 하나의 질문에 답하려면? — 팀 킥오프

> 3인 팀이 하나의 연구 질문을 확정하고, 분석 계획을 세울 수 있는가?

**학습 목표**
1. 팀 데이터를 확정하고, **연구 질문 1개와 가설(H0/H1)**을 수립할 수 있다.
2. 분석 계획(변수/척도/기법)과 **역할 분담**을 구체적으로 명세할 수 있다.
3. 팀원 각자의 기여를 **기록하는 습관**을 갖출 수 있다.

8주 과제에서 각자 조사한 데이터 후보를 팀원과 공유하고, 아래 기준으로 **팀 데이터 1개를 확정**한다.

**데이터 선택 기준**:
- 공개 데이터이거나 수업용으로 제공된 데이터인가?

### 무임승차 방지: 팀원별 기여 기록

매 체크포인트 제출 시 **팀원별 기여 3줄**을 반드시 포함한다. 이 기록은 개인 기여 평가(30%)의 근거가 된다.

**작성 형식**:
```
[이름1] 데이터 후보 2건 조사, 변수 정의 초안 작성, 팀 회의 진행
[이름2] 연구 질문 초안 작성, 가설 수립, 분석 기법 비교
[이름3] 데이터 탐색 코드 작성, 기술통계 정리, 회의록 작성
```

---

### 연구 질문과 가설 수립 가이드

좋은 연구 질문의 조건:
- **구체적**: "사회 현상에 대해 알아본다" (X) → "성별에 따라 SNS 이용 시간에 차이가 있는가?" (O)
- **검증 가능**: 데이터와 배운 기법으로 답할 수 있는 질문이어야 한다.
- **비인과적**: "A가 B를 일으키는가?"가 아니라 "A와 B 사이에 관련/차이가 있는가?"

가설 작성:
- **H0 (귀무가설)**: "차이가 없다" 또는 "관련이 없다"
- **H1 (대립가설)**: "차이가 있다" 또는 "관련이 있다"
