**Q.** "카이자승 검정에서 p값이 0.03이고 Cramer's V가 0.08이면, 이 결과를 어떻게 해석해야 하는가?"

**정답**: 통계적으로 유의(p = 0.03 < 0.05)하지만, Cramer's V = 0.08으로 효과 크기가 매우 약하다. 유의수준을 통과했더라도 실질적 의미는 크지 않다. 이는 표본 크기가 충분히 크면 아주 약한 연관도 유의하게 나올 수 있음을 보여주는 사례이다.

## 과제 1 기대 결과: 성별 x 지지정당

### 기대 코드

```python
import pandas as pd
from scipy.stats import chi2_contingency
import numpy as np

df = pd.read_csv("social_survey_w04.csv")

# 분할표 생성
ct = pd.crosstab(df["성별"], df["지지정당"], margins=True)
print("=== 분할표 (관찰빈도) ===")
print(ct)

# 카이자승 검정
ct_core = pd.crosstab(df["성별"], df["지지정당"])
chi2, p, dof, expected = chi2_contingency(ct_core)
print(f"\nχ²({dof}) = {chi2:.3f}, p = {p:.4f}")

# 기대빈도 확인
print("\n=== 기대빈도 ===")
print(pd.DataFrame(expected, index=ct_core.index,
                    columns=ct_core.columns).round(2))
print(f"최소 기대빈도: {expected.min():.2f}")

# Cramér's V
n = ct_core.sum().sum()
V = np.sqrt(chi2 / (n * (min(ct_core.shape) - 1)))
print(f"Cramér's V = {V:.3f}")
```

|---|---:|---:|---:|---:|---:|

| 여성 | 18 | 27 | 29 | 22 | 96 |
| 합계 | 63 | 43 | 48 | 46 | 200 |

**기대빈도**:

| | A당 | B당 | C당 | 무당파 |
|---|---:|---:|---:|---:|
| 남성 | 32.76 | 22.36 | 24.96 | 23.92 |
| 여성 | 30.24 | 20.64 | 23.04 | 22.08 |

**검정 결과**:

|---|---|
| chi-sq(df) | chi-sq(3) = 16.262 |
| p값 | 0.0010 |

## 모범 보고서 (A-E)

아래는 과제 1(성별 x 지지정당)을 선택한 경우의 모범 보고서이다.

### (A) 분석 질문

성별과 지지정당 사이에 통계적으로 유의한 연관이 있는가?

### (B) 사용 변수/척도

- 성별: 명목척도 (남성, 여성)
- 지지정당: 명목척도 (A당, B당, C당, 무당파)

### (C) 결과 근거

```python
import pandas as pd
from scipy.stats import chi2_contingency
import numpy as np

df = pd.read_csv("social_survey_w04.csv")
ct = pd.crosstab(df["성별"], df["지지정당"])
chi2, p, dof, expected = chi2_contingency(ct)
n = ct.sum().sum()
V = np.sqrt(chi2 / (n * (min(ct.shape) - 1)))
print(f"χ²({dof}) = {chi2:.3f}, p = {p:.4f}")
print(f"Cramér's V = {V:.3f}")
print(pd.DataFrame(expected, index=ct.index, columns=ct.columns).round(2))
```

- 분할표: 남성 A당 45 / B당 16 / C당 19 / 무당파 24, 여성 A당 18 / B당 27 / C당 29 / 무당파 22
- chi-sq(3) = 16.262, p = 0.0010
- 기대빈도 최솟값 20.64 (모든 셀 >= 5 충족)
- Cramer's V = 0.285

### (D) 해석문

**(분석목적)** 성별과 지지정당 사이에 통계적으로 유의한 연관이 있는지 확인하기 위해 카이자승 검정을 실시했다.

**(핵심근거)** 분할표에서 남성은 A당(43.3%), 여성은 B당(28.1%)과 C당(30.2%)을 상대적으로 더 지지하는 경향이 나타났다. chi-sq(3) = 16.262, p = 0.001이며, Cramer's V = 0.285이다. 기대빈도는 모든 셀에서 5 이상(최솟값 20.64)으로 검정 가정을 충족했다.

**(결론)** 유의수준 0.05에서 귀무가설(성별과 지지정당은 독립이다)을 기각한다. 두 변수 사이에 중간 수준의 연관이 관찰된다.

**(주의)** 이 결과는 성별이 정당 선택을 "결정"하거나 "원인"이라는 의미가 아니다. 연령, 교육수준, 거주지역 등 교란변수를 고려하지 않았으며, 설문 표본(n = 200)의 대표성에도 한계가 있다.

### (E) Copilot 사용 요약

Copilot에게 "성별과 지지정당의 분할표를 만들고 카이자승 검정을 실행해줘"라고 요청하여 기본 코드를 생성했다. Cramer's V 계산은 추가 프롬프트로 요청했다. 변수명 오타를 1회 수정했으며, Copilot이 생성한 해석문에서 "영향을 미친다"는 인과 표현을 "연관이 관찰된다"로 직접 수정했다.

## 흔한 오류 3개 + 확인법

### 오류 1: 기대빈도 확인 누락

`chi2_contingency()`는 4개의 값을 반환하며, 네 번째 값이 기대빈도 배열(`expected`)이다. 기대빈도를 출력하지 않고 p값만 보고하는 조가 많다.

**확인법**: `print(expected)`를 실행하여 모든 셀이 5 이상인지 눈으로 확인한다. 오늘 데이터에서는 과제 1 최솟값 20.64, 과제 2 최솟값 16.00으로 모두 충족한다.

### 오류 2: Cramer's V 미보고

p값만 보고하고 효과 크기를 보고하지 않는 경우가 흔하다. p값은 연관의 유무만 말해 줄 뿐 강도는 알려주지 않는다.

**확인법**: 결과 기록표에 Cramer's V가 비어 있다면 코드를 추가한다. 해석 기준: V < 0.1(약함), 0.1-0.3(중간), > 0.3(강함).

### 오류 3: 인과 표현 사용

"영향을 미친다", "결정한다", "~때문에" 등의 인과 표현이 해석문에 포함되는 경우가 가장 빈번하다.

**확인법**: 해석문(D항)에서 아래 단어를 검색하여 수정한다.
- "영향" -> "연관"
- "결정" -> "관계"
- "때문에" -> "에 따라"
- "원인" -> "차이"

## 비판적 읽기 + 토론 가이드

### 비판적 읽기 자료

아래 가상 뉴스 기사에서 문제점을 확인한다.

> **<OO일보> 2026.01.15**
>
> *"전국 성인 1,200명을 대상으로 한 설문 결과, 남녀의 정당 지지율이 다른 것으로 나타나, **성별이 정치 성향을 결정하는 주요 요인으로 확인되었다**. 이번 조사에서 남성의 52%가 A당을, 여성의 48%가 B당을 지지하는 것으로 드러났다."*

**문제점 3가지**:

1. **인과 표현**: "성별이 정치 성향을 **결정**하는 주요 요인"이라고 서술했다. 설문 데이터(횡단면 조사)로는 연관만 확인할 수 있으며, 인과를 주장하려면 실험 설계나 교란변수 통제가 필요하다. "성별에 따라 정당 지지율에 차이가 **관찰되었다**"가 적절하다.

2. **효과 크기 미보고**: 지지율 차이의 통계적 유의성(p값)이나 효과 크기(Cramer's V)가 전혀 보고되지 않았다. 1,200명이라는 큰 표본에서는 사소한 차이도 유의하게 나올 수 있으므로, 효과 크기 없이는 실질적 의미를 판단할 수 없다.

3. **교란변수 무시**: 연령, 교육수준, 소득, 거주지역 등 성별과 정치 성향 모두에 관련될 수 있는 변수가 고려되지 않았다. 이러한 교란변수를 통제하지 않은 상태에서 성별을 "주요 요인"이라고 단정하는 것은 과잉 해석이다.

### 토론 진행 가이드

아래 질문에 대한 답을 확인한다.

**질문 1**: "오늘 분석한 조합 중 Cramer's V가 가장 높은 것은 무엇이었는가?"
- 기대 답변: 연령대 x SNS이용여부 (V = 0.363)

**질문 2**: "유의하지만 V가 작으면 실질적 의미가 있다고 할 수 있는가?"
- 기대 답변: 통계적 유의성과 실질적 의미는 다르다. V가 0.1 미만이면 유의하더라도 실질적 의미는 크지 않을 수 있다. 빠른 진단의 V = 0.08 사례가 이에 해당한다.

**질문 3**: "유의하지 않은 결과(p > 0.05)는 '실패'인가?"
- 기대 답변: 아니다. 귀무가설을 기각하지 못했다는 것은 "두 변수가 독립이 아니라는 충분한 증거를 이 표본에서 찾지 못했다"는 뜻이지, "두 변수가 확실히 독립이다"는 뜻이 아니다. 이 또한 의미 있는 발견이다.

## Exit Ticket 모범 답안

**Q.** 카이자승 검정 결과 chi-sq(3) = 12.4, p = 0.006, Cramer's V = 0.12가 나왔다. 이 결과의 올바른 해석을 1-2문장으로 써라.

**모범 답안**: "유의수준 0.05에서 두 변수는 독립이 아닌 것으로 나타났다(chi-sq(3) = 12.4, p = 0.006). 그러나 Cramer's V = 0.12로 연관의 강도는 약한 수준이므로, 통계적으로 유의하더라도 실질적 의미는 크지 않을 수 있다."

**채점 포인트**:
- 유의성 판단 정확 (p < 0.05이므로 귀무가설 기각): 1점
- Cramer's V를 근거로 효과 크기 언급: 1점
- 유의성과 실질적 의미를 구별하는 표현: 1점
- 인과 표현 부재: 감점 요소 확인

## 해석문 모범 구조 (과제 2 선택 시)

과제 2(연령대 x SNS이용여부)를 선택한 조를 위한 모범 답안이다.

**(분석목적)** 연령대에 따라 SNS 이용 여부에 차이가 있는지 확인하기 위해 카이자승 검정을 실시했다.

**(핵심근거)** 분할표에서 20대의 SNS 이용률은 83.9%인 반면 50대 이상은 32.5%로, 연령대가 높을수록 이용률이 낮아지는 경향이 나타났다. chi-sq(3) = 26.355, p < 0.001이며, Cramer's V = 0.363이다. 기대빈도는 모든 셀에서 5 이상(최솟값 16.00)으로 검정 가정을 충족했다.
