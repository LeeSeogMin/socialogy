## 빠른 진단 정답

**Q.** "회귀 분석 결과 β₁ = 45.8, p < 0.001, R² = 0.285이다. '교육이 소득을 높인다'고 해석해도 되는가?"

**정답**: 해석할 수 없다. 회귀 분석은 두 변수의 연관/예측 관계를 보여줄 뿐, 인과를 증명하지 않는다. "교육 연수가 높을수록 소득이 높은 경향이 관찰된다"가 올바른 표현이다. 부모의 경제적 배경, 거주 지역, 직종 등 교란변수를 통제하지 않았으며, 역인과(소득이 높은 가정에서 교육 기간이 더 긴 경향)의 가능성도 존재한다.

## 과제 1 기대 결과: 교육연수 → 소득

### 기대 코드

```python
import pandas as pd
import statsmodels.api as sm
import matplotlib.pyplot as plt

df = pd.read_csv("social_survey_w07.csv")

# 산점도
plt.scatter(df["교육연수"], df["소득"], alpha=0.5)
plt.xlabel("교육연수")
plt.ylabel("소득(만원)")
plt.title("교육연수 vs 소득")
plt.show()

# 단순회귀
X = sm.add_constant(df["교육연수"])
model = sm.OLS(df["소득"], X).fit()
print(model.summary())

# 잔차 플롯
plt.scatter(model.fittedvalues, model.resid, alpha=0.5)
plt.axhline(y=0, color="red", linestyle="--")
plt.xlabel("예측값")
plt.ylabel("잔차")
plt.title("잔차 플롯")
plt.show()
```

### 기대 수치

**산점도**: 교육 연수가 높을수록 소득이 높아지는 양의 경향이 관찰된다. 점들이 오른쪽 위를 향해 흩어져 있다.

**회귀 결과**:

| 항목 | 값 |
|---|---|
| β₀(절편) | 85.2 |
| β₁(기울기) | 45.8 |
| β₁의 p값 | < 0.001 |
| R² | 0.285 |
| 잔차 패턴 | 패턴 없음 (무작위 분포) |
| 귀무가설 기각 여부 | 기각 (alpha = 0.05) |

**계수 해석**: 교육 연수가 1년 증가할 때, 소득은 평균 약 45.8만 원 증가하는 것으로 추정된다. 이 모형은 소득 변동의 약 28.5%를 교육 연수로 설명한다.

**잔차 해석**: 잔차 플롯에서 뚜렷한 패턴(깔때기 모양, U자 모양 등)은 관찰되지 않으며, 선형성 가정이 대체로 충족된 것으로 판단된다.

## 과제 2 기대 결과: 연령 → 생활만족도

### 기대 코드

```python
# 산점도
plt.scatter(df["연령"], df["생활만족도"], alpha=0.5)
plt.xlabel("연령")
plt.ylabel("생활만족도")
plt.title("연령 vs 생활만족도")
plt.show()

# 단순회귀
X2 = sm.add_constant(df["연령"])
model2 = sm.OLS(df["생활만족도"], X2).fit()
print(model2.summary())

# 잔차 플롯
plt.scatter(model2.fittedvalues, model2.resid, alpha=0.5)
plt.axhline(y=0, color="red", linestyle="--")
plt.xlabel("예측값")
plt.ylabel("잔차")
plt.title("잔차 플롯")
plt.show()
```

### 기대 수치

**산점도**: 연령과 생활만족도 사이에 뚜렷한 선형 경향이 보이지 않는다. 점들이 넓게 흩어져 있다.

**회귀 결과**:

| 항목 | 값 |
|---|---|
| β₀(절편) | 3.85 |
| β₁(기울기) | -0.012 |
| β₁의 p값 | 0.215 |
| R² | 0.008 |
| 잔차 패턴 | 패턴 없음 (무작위 분포) |
| 귀무가설 기각 여부 | 기각 못함 (alpha = 0.05) |

**계수 해석**: β₁ = -0.012이며 p = 0.215로, 유의수준 0.05에서 통계적으로 유의하지 않다. 연령이 1세 증가할 때 생활만족도가 0.012 감소하는 경향이 추정되지만, 이 관계는 표본 변동(우연)으로 설명될 수 있다.

**R² 해석**: R² = 0.008로, 이 모형은 생활만족도 변동의 0.8%만 설명한다. 연령은 생활만족도의 유의한 예측 변수가 아니다.

**참고**: "유의하지 않다"는 것은 "연령과 생활만족도 사이에 관계가 없다"는 뜻이 아니다. "이 표본에서 통계적으로 유의한 선형 관계의 증거를 찾지 못했다"는 뜻이다. 비선형 관계가 존재하거나, 표본 크기가 더 크면 결과가 달라질 수 있다.

## 모범 보고서 (A-E)

아래는 과제 1과 과제 2를 모두 포함한 모범 보고서이다.

### (A) 분석 질문

교육 연수는 소득을, 연령은 생활만족도를 통계적으로 유의하게 예측하는가?

### (B) 사용 변수/척도

- 교육연수: 비율척도 (6-18년, 연속형)
- 소득: 비율척도 (만 원, 연속형)
- 연령: 비율척도 (20-65세, 연속형)
- 생활만족도: 등간척도 (1-5점 리커트, 연속형으로 간주)

### (C) 결과 근거

```python
import pandas as pd
import statsmodels.api as sm
import matplotlib.pyplot as plt

df = pd.read_csv("social_survey_w07.csv")

# 과제 1: 교육연수 → 소득
X1 = sm.add_constant(df["교육연수"])
model1 = sm.OLS(df["소득"], X1).fit()
print("=== 과제 1: 교육연수 → 소득 ===")
print(model1.summary())

# 과제 2: 연령 → 생활만족도
```

- 과제 1: β₀ = 85.2, β₁ = 45.8(p < 0.001), R² = 0.285, 잔차 패턴 없음

### (D) 해석문

**(분석목적)** 교육 연수가 소득을, 연령이 생활만족도를 얼마나 예측하는지 확인하기 위해 각각 단순회귀 분석을 실행했다.

**(핵심근거)** 과제 1에서 β₁ = 45.8(p < 0.001), R² = 0.285로, 교육 연수가 1년 증가할 때 소득은 평균 약 45.8만 원 높은 것으로 추정된다. 과제 2에서 β₁ = -0.012(p = 0.215), R² = 0.008로, 연령과 생활만족도 사이에 통계적으로 유의한 선형 관계는 관찰되지 않았다. 두 모형 모두 잔차 플롯에서 뚜렷한 패턴은 보이지 않았다.

**(결론)** 유의수준 0.05에서, 교육 연수는 소득의 유의한 예측 변수이며 소득 변동의 약 28.5%를 설명한다. 반면, 연령은 생활만족도의 유의한 예측 변수가 아니다.

**(주의)** 이 결과는 교육이 소득을 "결정"하거나 연령이 만족도를 "변화시킨다"는 인과적 의미가 아니다. 부모의 경제적 배경, 직종, 건강 상태 등 교란변수를 통제하지 않았으며, 역인과의 가능성도 존재한다.

### (E) Copilot 사용 요약

Copilot에게 "교육연수와 소득의 산점도를 그리고 단순회귀를 실행해줘"라고 요청하여 기본 코드를 생성했다. 잔차 플롯은 추가 프롬프트로 요청했다. Copilot이 생성한 해석문에서 "교육이 소득을 높인다"는 인과 표현을 "교육 연수와 소득 사이에 양의 연관이 관찰된다"로 직접 수정했다.

## 흔한 오류 3개 + 확인법

### 오류 1: β₁을 인과로 해석

"교육이 소득에 영향을 미친다", "교육을 받으면 소득이 올라간다" 등의 인과 표현이 해석문에 포함되는 경우가 가장 빈번하다.

**확인법**: 해석문(D항)에서 아래 단어를 검색하여 수정한다.
- "영향" -> "연관"
- "높인다/올린다" -> "높은 것으로 추정된다"
- "때문에" -> "에 따라"
- "결정/원인" -> "예측 변수"

### 오류 2: R²만으로 모형의 가치를 단정

"R² = 0.285이라서 좋지 않다"고 결론짓거나, "R² = 0.008이라서 이 분석은 의미 없다"고 판단하는 경우가 많다.

**확인법**: R²는 반드시 맥락과 함께 해석한다. 사회과학에서 단일 변수 기준으로 R² = 0.285는 보통 수준의 설명력이다. R²가 낮더라도, "유의한 관계가 없다"는 결과 자체가 의미 있는 발견이다.

### 오류 3: 유의하지 않은 결과를 "실패"로 간주

과제 2에서 p = 0.215로 유의하지 않은 결과가 나오면, "분석을 잘못했다" 또는 "다른 변수를 찾아야 한다"고 반응하는 학생이 있다.

**확인법**: 유의하지 않은 결과는 "이 표본에서 연령과 생활만족도 사이에 통계적으로 유의한 선형 관계의 증거를 찾지 못했다"는 의미이며, 이 자체가 보고할 가치가 있는 결과이다. "관계가 없다"고 단정하는 것은 오류이다.

## 비판적 읽기 + 토론 가이드

### 비판적 읽기 자료

아래 가상 뉴스 기사에서 문제점을 확인한다.

> **<OO경제> 2026.03.10**
>
> *"한국노동패널 분석 결과, 교육 연수가 1년 증가할 때 월 소득은 평균 48만 원 증가하는 것으로 나타났다(R² = 0.31). 이는 **교육이 소득 향상의 핵심 원인**임을 보여주며, **학력을 높이면 확실히 소득이 올라간다**는 점을 입증한다."*

**문제점 3가지**:

1. **인과 표현**: "교육이 소득 향상의 핵심 **원인**"이라고 서술했다. 관찰 데이터의 회귀 분석으로는 연관/예측만 확인할 수 있다. "교육 연수와 소득 사이에 통계적으로 유의한 양의 연관이 **관찰되었다**"가 적절하다.

2. **역인과 가능성 무시**: 소득이 높은 가정에서 교육 기간이 더 길어질 수 있다. 이 가능성이 전혀 언급되지 않았다.

3. **교란변수 무시**: 부모의 경제적 배경, 사회적 네트워크, 거주 지역 등 교란변수가 고려되지 않았다. R² = 0.31이므로 소득 변동의 69%는 다른 요인으로 설명되어야 한다.

### 토론 진행 가이드

아래 질문에 대한 답을 확인한다.

**질문 1**: "과제 2에서 p = 0.215가 나왔다. 이것은 연령과 생활만족도가 '무관하다'는 뜻인가?"
- 기대 답변: 아니다. 귀무가설을 기각하지 못했다는 것은 "이 표본에서 통계적으로 유의한 선형 관계의 증거를 찾지 못했다"는 뜻이지, "두 변수가 확실히 무관하다"는 뜻이 아니다.

**질문 2**: "R²가 높으면 인과라고 할 수 있는가?"
- 기대 답변: 아니다. R²는 설명력(두 변수의 연관 강도)을 나타낼 뿐, 인과와는 별개의 개념이다. R² = 0.99라 해도 교란변수나 역인과가 있으면 인과라고 말할 수 없다.

**질문 3**: "잔차 플롯에서 깔때기 모양이 나타나면 어떻게 해야 하는가?"
- 기대 답변: 깔때기 모양은 이분산(heteroscedasticity)을 의미한다. 이 경우 표준오차가 부정확해질 수 있으므로, 강건 표준오차(robust standard errors)를 사용하거나, 변수 변환(로그 변환 등)을 고려할 수 있다.

## Exit Ticket 모범 답안

**Q.** 회귀 분석 결과 β₁ = -0.012, p = 0.215, R² = 0.008이 나왔다. 이 결과를 1-2문장으로 해석하라.

**모범 답안**: "유의수준 0.05에서 독립변수와 종속변수 사이에 통계적으로 유의한 선형 관계가 관찰되지 않았다(β₁ = -0.012, p = 0.215). R² = 0.008로 모형의 설명력이 매우 낮으며, 이 표본에서 해당 독립변수는 종속변수의 유의한 예측 변수가 아닌 것으로 나타났다."

**채점 포인트**:
- 유의성 판단 정확 (p > 0.05이므로 귀무가설 기각 못함): 1점
- R²를 근거로 설명력 언급: 1점
- "관계가 없다"고 단정하지 않고 "유의한 관계를 찾지 못했다"로 표현: 1점
- 인과 표현 부재: 감점 요소 확인
