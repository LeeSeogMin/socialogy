## A회차 빠른 진단 정답

**Q.** 단순회귀에서 교육연수의 β₁ = 45.8이었다. 경력연수를 추가한 다중회귀에서 교육연수의 β₁ = 38.4로 줄었다. 이 변화의 가장 적절한 해석은?

**정답**: ② 경력연수가 교육연수의 효과를 일부 "빼앗아" 설명하고 있다. 단순회귀에서 교육연수의 계수에는 경력연수의 효과가 일부 포함되어 있었으며, 경력을 모형에 추가하면 교육연수의 "순수한" 연관만 남게 되어 계수가 감소한다.

## B회차 빠른 진단 정답

**Q.** "단순회귀에서 교육연수 β₁ = 54.1이고, 경력연수를 추가한 다중회귀에서 β₁ = 38.4로 줄었다. 이 변화가 의미하는 바를 1문장으로 설명하라."

**정답**: 단순회귀에서 교육연수의 계수에는 경력연수의 효과가 일부 포함되어 있었으며, 경력연수를 모형에 추가하여 통계적으로 조정하면 교육연수의 "순수한" 연관만 남게 되므로 계수가 줄어든다. 이는 교육연수가 긴 사람은 경력도 함께 쌓이는 경향이 있기 때문이다.

## 과제 1 기대 결과: 단순회귀 → 다중회귀 계수 변화

### 기대 코드

```python
import pandas as pd
import statsmodels.api as sm

df = pd.read_csv("social_survey_w08.csv")

# 단순회귀: 소득 ~ 교육연수
X1 = sm.add_constant(df["교육연수"])
model1 = sm.OLS(df["소득"], X1).fit()
print("=== 단순회귀 ===")
print(model1.summary())

# 다중회귀: 소득 ~ 교육연수 + 경력연수
X2 = sm.add_constant(df[["교육연수", "경력연수"]])
model2 = sm.OLS(df["소득"], X2).fit()
print("\n=== 다중회귀 (2변수) ===")
print(model2.summary())
```

### 기대 수치

**단순회귀 (소득 ~ 교육연수)**:

| 항목 | 값 |
|---|---|
| β₀(절편) | 18.9 (참고값) |
| 교육연수 β₁ | 54.1 |
| 교육연수 p값 | < 0.001 |
| R² | 0.550 |

**다중회귀 (소득 ~ 교육연수 + 경력연수)**:

| 항목 | 값 |
|---|---|
| 교육연수 β₁ | 38.4 |
| 경력연수 β₂ | 12.3 |
| 교육연수 p값 | < 0.001 |
| 경력연수 p값 | < 0.001 |
| R² | 0.617 |

**계수 변화 해석**: 교육연수의 β₁이 54.1에서 38.4로 감소했다. 이는 단순회귀에서 교육연수의 계수에 경력연수의 효과가 포함되어 있었기 때문이다. 경력연수를 추가하여 통계적으로 조정하면, 교육연수의 "순수한" 연관(경력이 같은 수준인 사람끼리 비교했을 때의 교육 효과)만 남게 된다. R²도 0.550에서 0.617로 증가하여, 경력연수가 소득 변동의 추가적인 부분을 설명하고 있다.

## 과제 2 기대 결과: 3변수 다중회귀 + VIF

### 기대 코드

```python
from statsmodels.stats.outliers_influence import variance_inflation_factor

# 성별 더미변수 생성
df["성별_더미"] = (df["성별"] == "남성").astype(int)

# 다중회귀: 소득 ~ 교육연수 + 경력연수 + 성별_더미
X3 = sm.add_constant(df[["교육연수", "경력연수", "성별_더미"]])
model3 = sm.OLS(df["소득"], X3).fit()
print("=== 다중회귀 (3변수) ===")
print(model3.summary())

# VIF 확인
print("\n=== VIF ===")
for i, col in enumerate(X3.columns[1:]):
    print(f"{col}: VIF = {variance_inflation_factor(X3.values, i+1):.2f}")
```

### 기대 수치

**다중회귀 (소득 ~ 교육연수 + 경력연수 + 성별_더미)**:

| 항목 | 값 |
|---|---|
| 교육연수 β₁ | 36.8 |
| 경력연수 β₂ | 12.0 |
| 성별_더미 β₃ | 41.8 |
| 교육연수 p값 | < 0.001 |
| 경력연수 p값 | < 0.001 |
| 성별_더미 p값 | < 0.05 |
| R² | 0.625 |

**VIF 결과**:

| 변수 | VIF | 판단 |
|---|---:|---|
| 교육연수 | 1.75 | 문제 없음 (< 10) |
| 경력연수 | 1.70 | 문제 없음 (< 10) |
| 성별_더미 | 1.08 | 문제 없음 (< 10) |

**VIF 해석**: 세 변수의 VIF가 모두 2 미만으로, 다중공선성 문제는 없다. 각 독립변수가 서로 독립적인 정보를 제공하고 있으며, 계수 추정이 안정적이다.

**성별 계수 해석**: β₃ = 41.8은 "교육연수와 경력연수를 통계적으로 조정한 상태에서, 남성의 소득이 여성보다 평균 약 41.8만 원 높은 것으로 추정된다"는 의미이다. 이는 성별이 소득의 유의한 예측 변수임을 시사하지만, 성별이 소득 차이의 "원인"이라는 뜻이 아니다. 직종, 근무 형태 등 측정되지 않은 교란변수가 존재할 수 있다.

## 모범 보고서 (A-E)

아래는 과제 1, 과제 2, 과제 3을 모두 포함한 모범 보고서이다.

### (A) 분석 질문

경력연수와 성별을 통계적으로 조정하면 교육연수와 소득의 연관은 어떻게 달라지는가?

### (B) 사용 변수/척도

- 교육연수: 비율척도 (6-18년, 연속형)
- 경력연수: 비율척도 (0-40년, 연속형)
- 소득: 비율척도 (만 원, 연속형)
- 성별: 명목척도 (남성/여성 → 더미변수 변환)

### (C) 결과 근거

```python
import pandas as pd
import statsmodels.api as sm
from statsmodels.stats.outliers_influence import variance_inflation_factor

df = pd.read_csv("social_survey_w08.csv")
df["성별_더미"] = (df["성별"] == "남성").astype(int)

# 단순회귀
X1 = sm.add_constant(df["교육연수"])
model1 = sm.OLS(df["소득"], X1).fit()

# 다중회귀 (2변수)
X2 = sm.add_constant(df[["교육연수", "경력연수"]])
model2 = sm.OLS(df["소득"], X2).fit()

# 다중회귀 (3변수)
X3 = sm.add_constant(df[["교육연수", "경력연수", "성별_더미"]])
model3 = sm.OLS(df["소득"], X3).fit()

print("단순회귀:", model1.summary())
print("다중회귀(2변수):", model2.summary())
print("다중회귀(3변수):", model3.summary())

for i, col in enumerate(X3.columns[1:]):
    print(f"{col}: VIF = {variance_inflation_factor(X3.values, i+1):.2f}")
```

- 단순회귀: 교육연수 β₁ = 54.1(p < 0.001), R² = 0.550
- 다중회귀(2변수): 교육연수 β₁ = 38.4(p < 0.001), 경력연수 β₂ = 12.3(p < 0.001), R² = 0.617
- 다중회귀(3변수): 교육연수 β₁ = 36.8(p < 0.001), 경력연수 β₂ = 12.0(p < 0.001), 성별 β₃ = 41.8(p < 0.05), R² = 0.625
- VIF: 교육연수 1.75, 경력연수 1.70, 성별 1.08 (모두 10 미만)

### (D) 해석문

**(분석목적)** 경력연수와 성별을 통계적으로 조정했을 때 교육연수와 소득의 연관이 어떻게 달라지는지 확인하기 위해, 단순회귀와 다중회귀를 비교 실행했다.

**(핵심근거)** 단순회귀에서 교육연수 β₁ = 54.1이었으나, 경력연수를 추가한 다중회귀에서 β₁ = 38.4로, 성별까지 추가하면 β₁ = 36.8로 감소했다. R²는 0.550 → 0.617 → 0.625로 증가했다. VIF는 세 변수 모두 2 미만으로 다중공선성 문제는 없다.

**(결론)** 유의수준 0.05에서 교육연수, 경력연수, 성별은 모두 소득의 통계적으로 유의한 예측 변수이다. 변수를 추가할수록 교육연수의 계수가 감소하며, 이는 단순회귀에서 교육연수의 계수에 다른 변수의 효과가 섞여 있었음을 의미한다.

**(주의)** 이 결과는 교육이나 성별이 소득을 "결정"한다는 인과적 의미가 아니다. 관찰 데이터에서 "통제"는 통계적 조정에 불과하며, 부모의 경제적 배경, 직종, 거주 지역 등 측정되지 않은 교란변수가 여전히 존재할 수 있다.

### (E) Copilot 사용 요약

Copilot에게 "소득을 종속변수로, 교육연수를 독립변수로 단순회귀를 실행해줘"라고 요청하여 기본 코드를 생성했다. 다중회귀는 변수를 추가하여 동일한 방식으로 요청했다. VIF 계산 코드는 Copilot이 `variance_inflation_factor` 함수를 제안하여 활용했다. Copilot이 생성한 해석문에서 "교육이 소득에 영향을 미친다"는 인과 표현을 "교육연수와 소득 사이에 양의 연관이 관찰된다"로 직접 수정했다.

## 확장(Plus) 트랙 기대 결과

**표준화 계수 비교** (3변수 다중회귀):

표준화 계수(beta)는 각 변수를 표준화(평균 0, 표준편차 1)한 뒤 회귀를 실행하여 구한다. 변수의 단위가 다를 때 "어떤 변수가 Y에 더 강하게 연관되는가"를 비교할 수 있다.

| 변수 | 비표준화 β | 표준화 β (참고) |
|---|---:|---:|
| 교육연수 | 36.8 | 약 0.50 |
| 경력연수 | 12.0 | 약 0.33 |
| 성별_더미 | 41.8 | 약 0.10 |

**참고**: 비표준화 계수는 단위(교육: 년, 성별: 0/1)가 다르므로 직접 비교하기 어렵다. 표준화 계수로 비교하면 교육연수의 상대적 연관이 가장 큰 것으로 추정된다.

## 흔한 오류 3개 + 확인법

### 오류 1: "통제했다 = 인과를 증명했다"

"경력을 통제했으므로 교육이 소득의 원인이다"라고 해석하는 경우가 가장 빈번하다. 관찰 데이터에서의 통제는 통계적 조정일 뿐이며, 측정되지 않은 교란변수가 여전히 존재할 수 있다.

**확인법**: 해석문(D항)에서 아래 표현을 검색하여 수정한다.
- "통제했으므로 원인이다" → "통계적으로 조정한 상태에서 연관이 관찰된다"
- "인과를 확인했다" → "예측 관계를 확인했다"
- "영향을 미친다" → "유의한 예측 변수이다"

### 오류 2: 단순회귀 대비 계수 변화를 보고하지 않음

다중회귀만 실행하고, 단순회귀와 비교하지 않는 경우가 많다. 계수 변화는 "변수를 추가하면 해석이 어떻게 달라지는가"를 보여주는 핵심이다.

**확인법**: 단순회귀와 다중회귀의 교육연수 β₁을 반드시 나란히 기록하고, 변화 방향(증가/감소)과 그 이유를 1문장으로 서술한다.

### 오류 3: VIF 확인을 생략

다중공선성을 확인하지 않으면 계수의 신뢰성을 판단할 수 없다. VIF가 10 이상인 변수가 있으면 해당 변수를 제거하거나 결합하는 것을 고려해야 한다.

**확인법**: `variance_inflation_factor`를 사용하여 모든 독립변수의 VIF를 계산하고, 10 미만인지 확인한다.

## 비판적 읽기 + 토론 가이드

### 비판적 읽기 자료

아래 가상 연구 보고서에서 문제점을 확인한다.

> **<사회연구> 2026년 제42권**
>
> *"다중회귀 분석 결과, 교육 연수(β = 0.35, p < 0.001), 경력 연수(β = 0.22, p < 0.01), 성별(β = 0.18, p < 0.05)이 소득에 유의한 **영향을 미치는 것**으로 나타났다. 경력과 성별을 **통제함으로써** 교육이 소득의 **독립적 원인**임을 **확인**하였다."*

**문제점 3가지**:

1. **인과 표현**: "영향을 미치는 것", "독립적 원인"은 인과 표현이다. 관찰 데이터의 다중회귀로는 연관/예측만 확인할 수 있다. "교육연수는 소득의 유의한 예측 변수로 관찰되었다"가 적절하다.

2. **통제 = 인과 증명으로 오해**: "통제함으로써 원인임을 확인하였다"는 통제의 의미를 과대해석한 것이다. 통계적 조정은 측정된 변수에 한해서만 가능하며, 직종, 지역, 가정 배경 등 빠진 변수의 효과는 제거되지 않았다.

3. **측정되지 않은 교란변수 미언급**: 보고서에 모형의 한계가 전혀 서술되지 않았다. "부모의 경제적 배경, 직종 등 측정되지 않은 교란변수가 존재할 수 있다"는 한계를 반드시 명시해야 한다.

### 토론 진행 가이드

아래 질문에 대한 답을 확인한다.

**질문 1**: "단순회귀에서 교육연수 β₁ = 54.1인데 다중회귀에서 36.8로 줄었다. 단순회귀의 결과가 '틀린' 것인가?"
- 기대 답변: 틀린 것이 아니다. 단순회귀의 β₁은 "교육연수만 고려했을 때"의 연관이고, 다중회귀의 β₁은 "다른 변수를 통계적으로 조정한 후"의 연관이다. 질문의 성격이 다르며, 어느 쪽이 옳고 그른 것이 아니다.

**질문 2**: "변수를 많이 넣을수록 좋은 모형인가?"
- 기대 답변: 반드시 그렇지 않다. 이론적 근거 없이 변수를 무작위로 추가하면 과적합(overfitting)이 발생하거나, 다중공선성이 심해질 수 있다. 변수 선택은 이론 기반으로 하는 것이 원칙이다.

**질문 3**: "VIF가 3이면 문제인가?"
- 기대 답변: VIF 3은 일반적으로 문제로 보지 않는다. VIF > 5이면 주의가 필요하고, VIF > 10이면 심각한 공선성 문제로 판단한다. VIF 3 정도는 독립변수 간 약한 상관이 있다는 뜻이며, 계수 추정에 큰 영향을 주지 않는다.

## Exit Ticket 모범 답안

**Q.** 다중회귀에서 "경력연수를 통제한 상태에서 교육연수의 효과"라는 표현은, 실제로 경력연수를 고정한 실험을 했다는 뜻인가?

**모범 답안**: "실험을 한 것이 아니다. 관찰 데이터에서 '통제'는 통계적으로 경력연수의 효과를 조정(제거)했다는 뜻이지, 실제로 경력연수를 고정한 실험적 통제가 아니다. 따라서 측정되지 않은 교란변수가 여전히 존재할 수 있으며, 인과를 증명한 것이 아니다."

**채점 포인트**:
- "통계적 조정"과 "실험적 통제"의 구분: 1점
- 인과를 증명한 것이 아님을 명시: 1점
- 측정되지 않은 교란변수 가능성 언급: 1점
- "실험을 한 것이다"로 잘못 답하면 0점
