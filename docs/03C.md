**정답**: 표준편차(180만 원)가 평균(350만 원)의 절반 이상이므로, 소득 분포가 **매우 넓게 퍼져 있다**고 판단할 수 있다. A회차에서 배운 "표준편차가 평균에 비해 크면 이질적 집단" 기준을 적용한 것이다.

## 과제 1 기대 결과: 수치형 변수 기술통계 + 히스토그램

### 기대 코드

```python
import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("social_survey_w03.csv")

# 기술통계
desc = df[["연령", "소득", "생활만족도"]].describe()
print(desc)

# 히스토그램 (3개 변수)
fig, axes = plt.subplots(1, 3, figsize=(14, 4))
for i, col in enumerate(["연령", "소득", "생활만족도"]):
    df[col].hist(bins=15, ax=axes[i])
    axes[i].set_title(f"{col} 분포")
    axes[i].axvline(df[col].mean(), color='r', linestyle='--', label='평균')
    axes[i].axvline(df[col].median(), color='g', linestyle='-', label='중앙값')
    axes[i].legend(fontsize=8)
plt.tight_layout()
plt.show()
```

### 기대 수치

**표 W03.3 (정답)** 수치형 변수 기술통계 결과

| 변수 | 평균 | 중앙값 | 표준편차 | 평균-중앙값 차이 | 분포 형태 판정 |
|---|---|---|---|---|---|
| 연령 | 38.2 | 37.0 | 11.8 | +1.2 | **대체로 대칭** |
| 소득 | 348.5 | 285.0 | 182.3 | +63.5 | **우치우침** (평균 > 중앙값) |
| 생활만족도 | 3.18 | 3.0 | 1.05 | +0.18 | **대체로 대칭** |

### 해석 포인트

- **연령**: 평균과 중앙값의 차이가 1.2세로 작다. 히스토그램에서 비교적 고른 분포를 보인다.
- **소득**: 평균(348.5만 원)이 중앙값(285.0만 원)보다 63.5만 원 높다. 소수의 고소득자가 평균을 끌어올리는 **우측 치우침(right-skewed)** 분포이다. 표준편차(182.3만 원)가 매우 크므로 응답자 간 소득 격차가 크다.
- **생활만족도**: 평균과 중앙값의 차이가 0.18로 작아 대체로 대칭에 가깝다. 다만 서열 척도이므로 평균 해석에 유의해야 한다.

## 과제 2 기대 결과: 범주형 변수 빈도표 + 집단별 박스플롯

### 기대 코드

```python
import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("social_survey_w03.csv")

# 범주형 변수 빈도표
print("=== 교육수준 빈도 ===")
print(df["교육수준"].value_counts())
print(f"\n비율:\n{df['교육수준'].value_counts(normalize=True).round(3)}")

# 교육수준별 소득 요약
print("\n=== 교육수준별 소득 요약 ===")
print(df.groupby("교육수준")["소득"].agg(["mean", "median"]).round(1))

# 집단별 박스플롯: 교육수준별 소득
df.boxplot(column="소득", by="교육수준", figsize=(8, 5))
plt.title("교육수준별 소득 분포")
plt.suptitle("")
plt.ylabel("소득(만원)")
plt.show()
```

### 기대 수치

**표 W03.4 (정답)** 교육수준별 빈도 및 소득 요약

| 교육수준 | 빈도(명) | 비율(%) | 소득 평균 | 소득 중앙값 |
|---|---|---|---|---|
| 고졸이하 | 약 60 | 약 30% | 약 220 | 약 200 |
| 대졸 | 약 90 | 약 45% | 약 330 | 약 290 |
| 대학원이상 | 약 50 | 약 25% | 약 530 | 약 480 |

*(위 수치는 데이터 생성 시 설정에 따라 달라질 수 있다. 핵심 패턴이 동일한지 확인한다.)*

### 해석 포인트

- 박스플롯에서 **대학원이상 집단의 중앙값이 가장 높다**.
- 모든 집단에서 소득이 우측으로 치우쳐 있으며(상자 위쪽 수염이 더 길다), 특히 대졸 집단과 대학원이상 집단에서 **이상치**(상자 바깥의 점)가 관찰될 수 있다.
- 세 집단의 상자(Q1~Q3)가 일부 겹치지만, 대학원이상 집단은 다른 집단과 분리되는 경향이 있다.

## 흔한 오류 3개 + 확인법

### 오류 1: 범주형 변수에 평균을 구함

명목 척도(성별) 또는 서열 척도(교육수준)에 평균을 구하는 것은 부적절하다. 명목/서열 변수에는 **최빈값**과 **빈도표**가 적절한 요약 방법이다.

**확인법**: `.describe()`를 실행하기 전에 각 변수의 척도를 확인한다. 명목/서열 변수는 `.value_counts()`를 사용한다.

*(주의: 리커트 5점 척도(생활만족도)는 관행적으로 평균을 구하기도 하나, "서열 변수에 평균을 적용했다"는 한계를 반드시 명시해야 한다.)*

### 오류 2: 히스토그램의 bin 수를 너무 적게/많게 설정

bin이 너무 적으면(3-5개) 분포의 세부 형태가 사라지고, 너무 많으면(50개 이상) 노이즈처럼 보인다. **10-20개**가 대부분의 경우 적절하다.

**확인법**: `bins=15` 정도에서 시작하고, 분포 형태가 부자연스러우면 `bins=10` 또는 `bins=20`으로 조정한다. Copilot에게 "적절한 bin 수를 알려줘"라고 물어보는 것도 방법이다.

### 오류 3: 해석문에서 "주의" 단계를 빠뜨림

기술통계의 한계를 명시하지 않는 경우가 가장 흔하다. 아래 중 하나 이상을 반드시 포함한다.

**확인법**: 해석문을 작성한 후, 다음 3가지 한계 중 1개 이상이 포함되어 있는지 점검한다.
- **표본 한계**: 이 결과는 n=200의 표본에 한정되며, 모집단에 대한 추론이 아니다.
- **자기보고 한계**: 소득 등 자기보고 변수는 과소/과대 보고 가능성이 있다.
- **기술통계의 성격**: "있는 그대로의 요약"이지, 변수 간 인과관계를 말하지 않는다.

### 토론 포인트

- "소득 분포가 우치우침인데, 평균과 중앙값 중 어느 것이 더 대표적인가?"
  - **기대 답변**: 우치우침 분포에서는 중앙값이 평균보다 더 대표적이다. 소수의 고소득자가 평균을 끌어올리기 때문이다.

- "교육수준별 소득 차이가 관찰되었다. 이것이 '교육이 소득을 높인다'는 뜻인가?"
  - **기대 답변**: 아니다. 기술통계는 있는 그대로의 요약이다. 인과관계를 주장하려면 실험 설계나 통제변수가 필요하다. 여기서는 "교육수준에 따라 소득 분포에 차이가 관찰된다"고만 말할 수 있다.

- "해석문에서 가장 빠뜨리기 쉬운 단계는 무엇인가?"
  - **기대 답변**: "(주의) 한계" 단계이다. 학생들은 결론까지 쓰고 끝내는 경향이 있으므로, 한계 명시를 습관화해야 한다.

### 문제 발견 시 수정 방향

문제가 발견되면 아래 방향으로 수정한다.
- (분석목적)이 빠진 경우: "왜 이 분석을 했는지" 1문장을 추가한다

- (주의)가 빠진 경우: 표본 한계, 자기보고 한계, 기술통계의 성격 중 1개 이상 추가
