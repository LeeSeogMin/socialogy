## 빠른 진단 정답

**Q.** "t-검정 전에 Levene 검정 p = 0.02가 나왔다. 다음 단계로 무엇을 해야 하는가?"

**정답**: Levene 검정 p = 0.02 < 0.05이므로 등분산 가정이 위반되었다. 이 경우 일반 t-검정(Student's t) 대신 **Welch's t-검정**을 사용해야 한다. Python에서는 `stats.ttest_ind(group_a, group_b, equal_var=False)`로 실행한다.

## 과제 1 기대 결과: 성별 × 소득 (독립표본 t-검정)

### 기대 코드

```python
import pandas as pd
from scipy import stats
import numpy as np

df = pd.read_csv("social_survey_w06.csv")

male = df[df["성별"] == "남성"]["소득"]
female = df[df["성별"] == "여성"]["소득"]

# 기술통계
print("=== 기술통계 ===")
print(f"남성: n = {len(male)}, 평균 = {male.mean():.1f}, SD = {male.std():.1f}")
print(f"여성: n = {len(female)}, 평균 = {female.mean():.1f}, SD = {female.std():.1f}")

# 정규성 검정 (Shapiro-Wilk)
stat_m, p_m = stats.shapiro(male)
stat_f, p_f = stats.shapiro(female)
print(f"\n=== 정규성 검정 (Shapiro-Wilk) ===")
print(f"남성: W = {stat_m:.4f}, p = {p_m:.4f}")
print(f"여성: W = {stat_f:.4f}, p = {p_f:.4f}")

# 등분산 검정 (Levene)
lev_stat, lev_p = stats.levene(male, female)
print(f"\n=== 등분산 검정 (Levene) ===")
print(f"Levene: F = {lev_stat:.4f}, p = {lev_p:.4f}")

# 독립표본 t-검정
t_stat, p_val = stats.ttest_ind(male, female, equal_var=True)
print(f"\n=== t-검정 결과 ===")
print(f"t({len(male)+len(female)-2}) = {t_stat:.3f}, p = {p_val:.4f}")

# Cohen's d
n1, n2 = len(male), len(female)
pooled_std = np.sqrt(((n1-1)*male.std()**2 + (n2-1)*female.std()**2) / (n1+n2-2))
d = (male.mean() - female.mean()) / pooled_std
print(f"Cohen's d = {d:.3f}")
```

### 기대 수치

**기술통계**:

| 집단 | n | 평균(만 원) | SD |
|---|---:|---:|---:|
| 남성 | 100 | 355 | 165 |
| 여성 | 100 | 310 | 148 |

**가정 확인**:

| 검정 | 통계량 | p값 | 판정 |
|---|---|---|---|
| Shapiro-Wilk (남성) | - | p > 0.05 | 정규성 충족 |
| Shapiro-Wilk (여성) | - | p > 0.05 | 정규성 충족 |
| Levene 검정 | - | p = 0.42 | 등분산 충족 |

**참고**: 정규성 검정에서 p > 0.05이면 "정규분포가 아니라는 증거가 불충분하다"는 뜻이다. 정규분포임을 증명하는 것이 아니라, 정규성을 기각할 근거가 없다는 것이다.

**검정 결과**:

| 항목 | 값 |
|---|---|
| t(df) | t(198) = 2.45 |
| p값 | 0.015 |
| Cohen's d | 0.35 |
| 효과 크기 판정 | 작은~중간 (small-medium) |
| 귀무가설 기각 여부 | 기각 (alpha = 0.05) |

**해석**: 남성의 평균 소득(355만 원)이 여성(310만 원)보다 약 45만 원 높으며, 이 차이는 유의수준 0.05에서 통계적으로 유의하다. Cohen's d = 0.35로 작은~중간 수준의 효과이다.

## 과제 2 기대 결과: 교육수준 × 소득 (ANOVA)

### 기대 코드

```python
# 집단별 기술통계
for group in ["고졸", "대졸", "대학원"]:
    subset = df[df["교육수준"] == group]["소득"]
    print(f"{group}: n = {len(subset)}, 평균 = {subset.mean():.1f}, SD = {subset.std():.1f}")

# 등분산 검정 (Levene)
high = df[df["교육수준"] == "고졸"]["소득"]
college = df[df["교육수준"] == "대졸"]["소득"]
grad = df[df["교육수준"] == "대학원"]["소득"]

lev_stat, lev_p = stats.levene(high, college, grad)
print(f"\nLevene: F = {lev_stat:.4f}, p = {lev_p:.4f}")

# ANOVA
f_stat, p_val = stats.f_oneway(high, college, grad)
df_between = 2
df_within = len(df) - 3
print(f"\nF({df_between}, {df_within}) = {f_stat:.2f}, p = {p_val:.6f}")

# η² (eta squared)
ss_between = sum(len(g) * (g.mean() - df["소득"].mean())**2
                 for g in [high, college, grad])
ss_total = sum((df["소득"] - df["소득"].mean())**2)
eta_sq = ss_between / ss_total
print(f"η² = {eta_sq:.3f}")

# Tukey 사후검정
from statsmodels.stats.multicomp import pairwise_tukeyhsd
tukey = pairwise_tukeyhsd(df["소득"], df["교육수준"], alpha=0.05)
print(tukey)
```

### 기대 수치

**기술통계**:

| 집단 | n | 평균(만 원) | SD |
|---|---:|---:|---:|
| 고졸 | 약 67 | 238 | - |
| 대졸 | 약 67 | 332 | - |
| 대학원 | 약 66 | 498 | - |

**가정 확인**:

| 검정 | p값 | 판정 |
|---|---|---|
| Levene 검정 | p = 0.08 | 등분산 충족 (p > 0.05) |

**참고**: Levene p = 0.08은 유의수준 0.05에서 등분산 가정을 기각하지 못한다. 그러나 0.05에 가까운 값이므로, 결과 해석 시 주의가 필요하다.

**검정 결과**:

| 항목 | 값 |
|---|---|
| F(df1, df2) | F(2, 197) = 18.75 |
| p값 | < 0.001 |
| η² | 0.160 |
| 효과 크기 판정 | 큰 효과 (large, η² > 0.14) |
| 귀무가설 기각 여부 | 기각 (alpha = 0.05) |

**Tukey 사후검정 결과**:

| 집단 쌍 | 평균 차이 | p값 | 유의 여부 |
|---|---:|---|---|
| 고졸 - 대졸 | 94만 원 | p = 0.002 | 유의 |
| 고졸 - 대학원 | 260만 원 | p < 0.001 | 유의 |
| 대졸 - 대학원 | 166만 원 | p < 0.001 | 유의 |

**해석**: 교육수준에 따라 평균 소득에 통계적으로 유의한 차이가 있다(F(2, 197) = 18.75, p < 0.001). η² = 0.160으로 큰 효과이다. Tukey 사후검정 결과, **세 집단 모두 서로** 유의한 차이가 나타났다. 고졸(238만 원) < 대졸(332만 원) < 대학원(498만 원) 순으로 평균 소득이 높았다.

## 모범 보고서 (A-E)

아래는 과제 1(성별 × 소득)을 선택한 경우의 모범 보고서이다.

### (A) 분석 질문

성별에 따라 평균 소득에 통계적으로 유의한 차이가 있는가?

### (B) 사용 변수/척도

- 성별: 명목척도 (남성, 여성) — 독립변수
- 소득: 비율척도 (단위: 만 원) — 종속변수

### (C) 결과 근거

```python
from scipy import stats
import pandas as pd
import numpy as np

df = pd.read_csv("social_survey_w06.csv")
male = df[df["성별"] == "남성"]["소득"]
female = df[df["성별"] == "여성"]["소득"]

# 가정 확인
print(f"Shapiro 남성: p = {stats.shapiro(male)[1]:.4f}")
print(f"Shapiro 여성: p = {stats.shapiro(female)[1]:.4f}")
lev_stat, lev_p = stats.levene(male, female)
print(f"Levene: p = {lev_p:.4f}")

# t-검정
t_stat, p_val = stats.ttest_ind(male, female)
n1, n2 = len(male), len(female)
pooled_std = np.sqrt(((n1-1)*male.std()**2 + (n2-1)*female.std()**2) / (n1+n2-2))
d = (male.mean() - female.mean()) / pooled_std
print(f"t({n1+n2-2}) = {t_stat:.3f}, p = {p_val:.4f}, Cohen's d = {d:.3f}")
```

- 기술통계: 남성 평균 355만 원(SD 165), 여성 평균 310만 원(SD 148)
- 정규성: Shapiro-Wilk 남성 p > 0.05, 여성 p > 0.05 (충족)
- 등분산: Levene p = 0.42 (충족)
- t(198) = 2.45, p = 0.015
- Cohen's d = 0.35

### (D) 해석문

**(분석목적)** 성별에 따라 평균 소득에 차이가 있는지 확인하기 위해 독립표본 t-검정을 실시했다.

**(핵심근거)** 정규성(Shapiro-Wilk: 남성 p > 0.05, 여성 p > 0.05)과 등분산(Levene p = 0.42) 가정이 모두 충족되었다. t(198) = 2.45, p = 0.015이며, Cohen's d = 0.35이다. 남성 평균 소득은 355만 원(SD 165), 여성 평균 소득은 310만 원(SD 148)이다.

**(결론)** 유의수준 0.05에서 귀무가설(남성과 여성의 평균 소득이 같다)을 기각한다. 남성의 평균 소득이 여성보다 약 45만 원 높으며, 작은~중간 수준의 효과(d = 0.35)이다.

**(주의)** 이 결과는 성별이 소득을 "결정한다"거나 "원인"이라는 의미가 아니다. 직종, 경력, 근무시간, 교육수준 등 교란변수를 통제하지 않았으며, 가상 데이터(n = 200)의 표본 한계도 있다.

### (E) Copilot 사용 요약

Copilot에게 "성별에 따른 소득의 t-검정을 가정 확인 포함하여 실행해줘"라고 요청하여 기본 코드를 생성했다. Cohen's d 계산은 추가 프롬프트로 요청했다. Copilot이 생성한 해석문에서 "성별이 소득에 영향을 미친다"는 인과 표현을 "성별에 따라 소득에 차이가 관찰된다"로 직접 수정했다.

## 흔한 오류 3개 + 확인법

### 오류 1: 가정 확인 누락

t-검정이나 ANOVA를 실행하기 전에 정규성(Shapiro-Wilk)과 등분산(Levene) 검정을 건너뛰는 경우가 흔하다. 가정이 위반되면 검정 결과의 신뢰성이 떨어진다.

**확인법**: 코드에 Shapiro-Wilk과 Levene 검정이 포함되어 있는지 확인한다. 오늘 데이터에서는 과제 1의 Levene p = 0.42(충족), 과제 2의 Levene p = 0.08(충족)이다.

### 오류 2: ANOVA 유의 → "모든 집단이 다르다"

ANOVA의 F-검정이 유의하면 "적어도 하나의 집단 쌍이 다르다"만 말해 준다. 어느 쌍이 다른지는 사후검정으로 확인해야 한다. 오늘 데이터에서는 세 쌍 모두 유의하지만, 항상 그런 것은 아니다.

**확인법**: ANOVA가 유의한 경우 Tukey HSD 결과가 포함되어 있는지, 그리고 해석문에서 "모든 집단이 다르다"가 아닌 "사후검정 결과 ___쌍에서 유의한 차이가 나타났다"로 서술했는지 확인한다.

### 오류 3: 효과 크기 미보고

p값만 보고하고 Cohen's d나 η²를 보고하지 않는 경우가 빈번하다. "유의하다 = 효과가 크다"는 오해를 방지하기 위해 효과 크기를 반드시 함께 보고한다.

**확인법**: 결과 기록표에 Cohen's d(과제 1) 또는 η²(과제 2)가 비어 있다면 코드를 추가한다. 해석 기준: Cohen's d(0.2/0.5/0.8), η²(0.01/0.06/0.14).

## Exit Ticket 모범 답안

**Q.** ANOVA 결과 F(2, 197) = 5.23, p = 0.006, η² = 0.05가 나왔다. 이 결과만 가지고 "세 집단의 평균이 모두 서로 다르다"고 결론 내릴 수 있는가? 이유를 1-2문장으로 써라.

**모범 답안**: "결론 내릴 수 없다. ANOVA의 F-검정은 '적어도 하나의 집단 쌍에서 유의한 차이가 있다'만 알려줄 뿐, 어느 쌍이 다른지는 사후검정(Tukey HSD 등)으로 확인해야 한다."

**채점 포인트**:
- ANOVA가 "적어도 하나의 쌍"만 알려준다는 점 언급: 1점
- 사후검정이 필요하다는 점 언급: 1점
- "모든 집단이 다르다"고 단정할 수 없다는 판단: 1점
