# W10. 질문부터 해석까지 혼자 완주할 수 있는가? — 개인 미니 완주

> 1-8주에서 배운 분석 흐름을 처음부터 끝까지 혼자 수행하여, 1쪽 요약 메모를 완성할 수 있는가?

**학습 목표**
1. 연구 질문 → 변수 정의 → 기술통계 → 검정 선택 → 가정 확인 → 실행 → 해석문의 **전 과정을 독립적으로 수행**할 수 있다.
2. 분석 결과를 사회과학 글쓰기 구조(서론 → 방법 → 결과 → 논의)에 맞추어 **1쪽 요약 메모**로 정리할 수 있다.
3. 자신의 분석 과정을 되돌아보며, **한계와 개선점**을 식별할 수 있다.

---

## A회차: 분석 흐름 완주 안내

### 이번 주의 위치

1-8주에서 다룬 내용을 정리하면 다음과 같다.

| 주차 | 핵심 기법 |
|---:|---|
| 2주 | 데이터 리터러시, 정제 |
| 3주 | 기술통계, 분포, 산포 |
| 4주 | 분할표, 카이자승 검정 |
| 5주 | 가설검정 논리 |
| 6주 | t-검정, ANOVA |
| 7주 | 단순회귀 |
| 8주 | 중다회귀 |

이번 주는 **새로운 개념 없이**, 위 기법을 연구 질문 하나에 통합 적용하는 시간이다. 교수가 제시한 연구 질문 2-3개 중 1개를 골라, 전 과정을 스스로 수행한다.

---

### 분석 흐름 완주 체크리스트

모든 분석은 아래 7단계를 따른다. 과제 수행 시 이 체크리스트를 순서대로 점검한다.

| 단계 | 내용 | 확인 |
|---:|---|---|
| 1 | **연구 질문** 설정 (1문장) | [ ] |
| 2 | **변수 정의**: 독립/종속 변수, 척도(명목/서열/등간·비율) 명시 | [ ] |
| 3 | **기술통계**: 빈도표, 평균/표준편차, 히스토그램 등으로 데이터 탐색 | [ ] |
| 4 | **검정 선택**: 변수 유형과 질문에 맞는 분석 기법 선택 (아래 표 참고) | [ ] |
| 5 | **가정 확인**: 선택한 검정의 전제 조건 충족 여부 확인 | [ ] |
| 6 | **검정 실행**: 코드 실행 및 결과 확인 | [ ] |
| 7 | **해석문 작성**: 4단계 해석문 (분석목적 → 핵심근거 → 결론 → 주의/한계) | [ ] |

---

### 검정 선택 가이드

| 독립변수 | 종속변수 | 추천 기법 |
|---|---|---|
| 범주형 (2수준) | 범주형 | 카이자승 검정 |
| 범주형 (2수준) | 연속형 | 독립표본 t-검정 |
| 범주형 (3수준+) | 연속형 | 일원분산분석(ANOVA) |
| 연속형 | 연속형 | 단순회귀/중다회귀 |

---

### 1쪽 요약 메모 구조

최종 제출물은 아래 구조의 1쪽 요약 메모이다. 사회과학 글쓰기의 기본 구조를 따른다.

1. **서론** (2-3문장): 연구 질문과 그 질문이 왜 중요한지
2. **방법** (3-4문장): 사용 데이터, 변수(척도 포함), 분석 기법, 가정 확인 결과
3. **결과** (3-4문장): 핵심 수치(검정통계량, p값, 효과크기 등) + 표/그래프 1개
4. **논의** (2-3문장): 결론, 한계, "이 분석이 말할 수 없는 것"

---

### Copilot 활용 안내

이번 과제에서도 Copilot을 디버깅·코드 생성·해석 초안 도우미로 활용할 수 있다. 단, **최종 해석문은 본인이 직접 작성**한다.

**추천 프롬프트**:
- "이 데이터에서 [변수A]와 [변수B]의 관계를 분석하려면 어떤 검정을 사용해야 해? 변수A는 [척도], 변수B는 [척도]야."
- "아래 코드를 실행했더니 [오류 메시지]가 나왔어. 원인과 해결법을 알려줘."

## B회차: 개인 미니 종합 — 과제 9

### 과제 안내

| 항목 | 내용 |
|---|---|
| 과제 유형 | **개인 과제** (조별 아님) |
| 소요 시간 | 40분 |
| 제출 형식 | A-E 형식 + 1쪽 요약 메모 |
| 제출처 | Google Classroom |

---

### 과제 구성

#### ① 연구 질문 선택 및 변수 정의

교수가 제시한 연구 질문 2-3개 중 **1개를 선택**한다. 선택 후 아래를 작성한다.

- (A) 분석 질문 (1-2문장)
- (B) 사용 변수와 척도 (명목/서열/등간·비율) + 선택한 검정 기법과 그 이유

#### ② 분석 실행 및 결과 확인

분석 흐름 완주 체크리스트의 3-6단계를 수행한다.

- 기술통계로 데이터를 탐색한다.
- 선택한 검정의 가정을 확인한다.
- 검정을 실행하고 결과를 확인한다.
- (C) 핵심 표/그래프 1-2개 또는 수치 3개 이내를 정리한다.

#### ③ 해석문 및 요약 메모 작성

- (D) 해석문 (5-7문장): 4단계 구조를 따른다.
  - "(분석목적) ___를 보기 위해 ___를 사용했다."
  - "(핵심근거) 출력에서 ___(검정통계량/df/p값)을 확인했다."
  - "(결론) 유의수준 ___에서 ___를 ___(기각/채택)한다."
  - "(주의) 이는 ___를 의미하지 않으며, ___의 한계가 있다."
- (E) Copilot 사용 요약 (2-3줄): 어떤 요청을 했고, 무엇을 검증했는가
- **1쪽 요약 메모**: 서론-방법-결과-논의 구조로 정리한다.

---

### 기본(Basic) / 확장(Plus) 트랙

| 트랙 | 내용 |
|---|---|
| **기본** (필수) | 교수 제시 질문 1개를 분석하고, A-E + 1쪽 요약 메모 제출 |
| **확장** (가산점) | 본인이 직접 설정한 연구 질문으로 추가 분석 1건 수행 |

---

### 흔한 오류 3가지

| 오류 | 확인법 |
|---|---|
| 변수 척도를 잘못 판단하여 부적절한 검정을 선택한다 | 검정 선택 가이드 표를 다시 확인한다 |
| 가정 확인을 생략하고 바로 검정을 실행한다 | 체크리스트 5단계를 반드시 거친다 |
| 해석문에서 "인과"를 주장한다 | "관련이 있다/차이가 있다"로 표현하고, "~때문이다"는 사용하지 않는다 |
