# W10. 공공 데이터에서 나의 연구 질문을 찾을 수 있는가? — 개인 리서치 브리프 킥오프

> 개인별로 공공 마이크로데이터를 선택하고, 본인의 관심 주제에서 검증 가능한 연구 질문 1개를 확정할 수 있는가?

**학습 목표**
1. 한국의 공공 마이크로데이터 6종의 성격을 이해하고, 본인 관심 주제에 맞는 데이터 1개를 선택할 수 있다.
2. 선택한 데이터에서 1-8주에 배운 기법으로 답할 수 있는 연구 질문 1개를 수립할 수 있다.
3. 해당 주제의 선행연구 2편(한국어 1편·영어 1편)을 찾아 본인 연구의 위치를 서술할 수 있다.

---

## A회차: 개인 리서치 브리프 안내

### 이번 주부터의 변화: 팀 프로젝트 → 개인 리서치 브리프

9주 중간고사 이후, 이번 학기 후반부는 **개인별 리서치 브리프(6-8쪽)** 를 작성하는 시간이다. 1-8주에서 배운 기법을 본인이 직접 선택한 연구 질문에 적용하여, 사회과학 분석 보고서 한 편을 완주한다.

**"리서치 브리프"란 무엇인가**
- 전문 학술논문(소논문)이 아니다.
- 공개 데이터를 분석하여 하나의 사회학적 질문에 답하는 **단일 저자 분석 보고서**이다.
- 분량 6-8쪽(A4), 참고문헌 최소 5편.
- 학부생이 한 학기 안에 완주 가능한 범위로 설계되어 있다.

### 추천 공공 마이크로데이터 풀

아래 6종 중 **1개를 선택**한다. 모두 공개되어 있으며 학부 수준에서 분석 가능하다.

| 데이터 | 주제 영역 | 접근 | 특징 |
|---|---|---|---|
| **KGSS (한국종합사회조사)** | 사회 전반(가치관·정치·노동·가족) | kgss.skku.edu | 미국 GSS의 한국판, 사회학 학부 표준 |
| **KOWEPS (한국복지패널)** | 빈곤·불평등·복지 | koweps.re.kr | 종단 자료, 횡단면 추출 가능 |
| **KELS (한국교육종단연구)** | 교육·학업성취·진로 | kedi.re.kr | 학생 패널, 교육사회학 주제 |
| **청소년정책연구원 패널** | 청소년·진학·가족 | nypi.re.kr | 다년 추적, 풍부한 변수 |
| **지역사회건강조사** | 건강행동·의료이용 | chs.kdca.go.kr | 지역 단위 비교 가능 |
| **KOSIS 마이크로데이터** | 경제·인구·사회 전반 | mdis.kostat.go.kr | 통계청 운영, 다양한 주제 |

**대안**: 본인이 KOSSDA(kossda.snu.ac.kr) 또는 공공데이터포털에서 직접 발굴해도 된다(가산점). 단, 학술적 인용이 가능한 데이터여야 하며, 10주 제출 시 승인받아야 한다.

**금지**: 본인이 직접 설문을 돌린 데이터는 사용할 수 없다(IRB·표본·품질 문제). 기업 내부 데이터도 제외한다.

**각 데이터의 접근 절차·코드북·pandas 로딩 방법**은 `docs/data_pool_guide.md`를 참고한다. 모든 데이터는 회원가입과 이용 신청 후 승인까지 1-3일 소요되므로 **10주차 이전에 미리 신청**해 두어야 한다.

---

### 좋은 연구 질문의 3조건

1. **구체적**: "세대 간 사회이동을 본다" (X) → "부모의 학력에 따라 30대 자녀의 취업 형태(정규직/비정규직)에 차이가 있는가?" (O)
2. **검증 가능**: 1-8주 기법(카이제곱·t·ANOVA·회귀)으로 답할 수 있어야 한다.
3. **비인과적**: "A가 B를 야기하는가?"가 아니라 "A와 B 사이에 관련/차이가 있는가?"

**주의**: 관심이 아무리 커도 **배운 기법으로 답할 수 없는 질문**은 고를 수 없다. 연구 질문은 데이터와 기법에 맞게 재단하는 작업이다.

---

### 선행연구 찾기 — 최소 2편

- **한국어 1편**: RISS(riss.kr), DBpia(dbpia.co.kr), KISS(kiss.kstudy.com) 등에서 검색.
- **영어 1편**: Google Scholar, JSTOR, Sociological Abstracts 등에서 검색.
- 학위논문은 최대 1편까지만 허용(학술지 논문 우선).

**Copilot 활용 예시**:
- "사회학 분야에서 [관심주제]에 관한 최근 10년 내 한국어 학술지 논문 검색 키워드를 제안해줘."
- "이 논문 초록을 읽고, 내 연구 질문과의 연관성을 3줄로 요약해줘."

**주의**: Copilot이 생성한 참고문헌은 자주 **허위 인용(hallucination)** 이다. 반드시 원문 DOI/페이지를 직접 확인하고 서지정보를 복사해야 한다.

### 제출물 구성

**제출 템플릿**: `practice/brief/cp1_data_question.md`을 복사하여 작성한다.

#### ① 데이터 선택

- 선택한 데이터 이름과 URL
- 표본 크기(N) 및 조사 연도
- 선택 이유 (2-3문장)

#### ② 연구 질문과 가설

- 연구 질문 1개 (1-2문장)
- H0 / H1
- 예상 결과와 그 의미 (2-3문장)

#### ③ 변수 명세

아래 표 형식으로 제출한다.

| 역할 | 변수명 | 데이터 변수 코드 | 척도 | 측정 방식 |
|---|---|---|---|---|
| 독립변수 | | | | |
| 종속변수 | | | | |
| 통제변수 1 | | | | |
| 통제변수 2 | | | | |

통제변수는 중다회귀 준비를 위해 **최소 2개** 선정한다.

#### ④ 선행연구 2편

- 한국어 1편, 영어 1편 서지정보(APA)
- 각 논문을 3-4줄로 요약
- 본 연구와의 연관성 1-2문장

#### ⑤ 분석 계획

- 1차 검정(카이제곱/t/ANOVA 중) 기법과 선택 이유
- 예상되는 가정 확인 항목

---

### 흔한 오류 3가지

| 오류 | 확인법 |
|---|---|
| 관심 주제만 있고, 변수로 조작화되지 않는다 | 데이터 코드북에서 실제 변수명과 코딩을 확인한다 |
| 연구 질문이 1-8주 기법으로 답할 수 없다 | 검정 선택 가이드(W10 상단)로 돌아간다 |
| 선행연구를 Copilot에게 생성하게 한다 | 반드시 원문 검색 사이트에서 직접 확인한다 |
