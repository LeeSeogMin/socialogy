## 핵심 개념 학습

### 왜 필요한가

4주차에서는 범주형 변수 간 관계를 분석했다(카이자승 검정). 그런데 사회과학에서는 **연속형 변수**(소득, 점수, 만족도 등)의 집단 간 차이를 비교하는 질문이 매우 흔하다. 이 질문에 답하기 위한 도구가 바로 **t-검정**과 **ANOVA**이다.

---

### 핵심 개념 1: 독립표본 t-검정(independent samples t-test)

**독립표본 t-검정(independent samples t-test)**은 두 독립 집단의 평균이 통계적으로 유의하게 다른지를 검정하는 방법이다. 핵심 직관은 다음과 같다.

> t = (두 집단의 평균 차이) ÷ (그 차이의 표준오차)

쉽게 말해서, **분자**(평균 차이)가 크거나 **분모**(변동)가 작으면 t값이 커지고, 유의할 가능성이 높아진다.

**t-검정의 세 가지 가정:**

1. **독립성**: 두 집단의 관찰값이 서로 독립적이어야 한다.
2. **정규성**: 각 집단의 분포가 정규분포에 가까워야 한다. n ≥ 30이면 중심극한정리로 크게 문제되지 않지만, n < 30이면 **Shapiro-Wilk 검정**으로 확인한다.
3. **등분산(homogeneity of variance)**: 두 집단의 분산이 같아야 한다. **Levene 검정**으로 확인하며, 위반 시 **Welch's t-검정**을 사용한다.

---

#### 예측 투표 — 가정 확인 순서

> **투표 질문**: t-검정을 실행하기 전에 가정을 확인해야 한다. 다음 중 올바른 확인 순서는?
>
> (가) 등분산 → 정규성 → 독립성
> (나) 독립성 → 정규성 → 등분산
> (다) 정규성 → 독립성 → 등분산
>
> 보자. (10초)

정답은 **(나)**이다. **독립성**은 연구 설계에서 먼저 판단하고(같은 사람이 두 번 응답하지 않았는지), **정규성**(Shapiro-Wilk)을 검정한 뒤, **등분산**(Levene)을 확인한다. 이 순서가 가정 확인 체크리스트이다.

**가정 확인 체크리스트:**

| 순서 | 가정 | 확인 방법 | 위반 시 대안 |
|---:|---|---|---|
| 1 | 독립성 | 연구 설계 확인 | 대응표본 t-검정 |
| 2 | 정규성 | Shapiro-Wilk 검정 (n < 30) | Mann-Whitney U |
| 3 | 등분산 | Levene 검정 | Welch's t-검정 |

---

### 핵심 개념 2: ANOVA(Analysis of Variance)

**ANOVA(Analysis of Variance, 분산분석)**는 세 집단 이상의 평균이 모두 같은지를 한꺼번에 검정하는 방법이다. 귀무가설은 "모든 집단의 모평균이 동일하다"이며, 이를 기각하면 "적어도 하나의 집단 평균이 다르다"고 판단한다.

여기서 중요한 질문이 있다: **"왜 t-검정을 여러 번 하면 안 되는가?"** 3개 집단을 비교할 때 t-검정을 3번 실행하면, 전체 1종 오류율이 약 14.3%까지 올라간다. 이를 **다중비교 문제(multiple comparison problem)**라고 하며, ANOVA는 한 번의 F-검정으로 이 문제를 해결한다.

> F = 집단 간 변동(between) ÷ 집단 내 변동(within)

집단 간 차이가 집단 내 변동에 비해 충분히 크면 F값이 커지고, 귀무가설을 기각한다.

---

### 핵심 개념 3: 사후검정(post-hoc test)

ANOVA에서 F-검정이 유의하면 "적어도 하나의 집단 쌍이 다르다"는 것만 알 수 있다. **어느 집단 쌍이 다른지**는 **사후검정(post-hoc test)**으로 확인한다.

- **Tukey HSD(Honestly Significant Difference)**: 모든 집단 쌍을 비교하면서 다중비교 보정을 적용한다.
- **Bonferroni 보정**: 유의수준(α)을 비교 횟수로 나누어 더 엄격한 기준을 적용한다.

> **주의**: ANOVA가 유의하다고 해서 "**모든** 집단이 서로 다르다"는 뜻이 아니다.

---

#### 오답 수집 — "ANOVA 유의 = 모든 집단이 다르다?"

> **질문**: ANOVA 결과 F(2, 197) = 18.75, p < 0.001이 나왔다. 다음 중 올바른 해석은?
>
> ① 세 집단의 평균이 모두 서로 다르다
> ② 적어도 하나의 집단 쌍에서 평균 차이가 유의하다
> ③ 첫 번째 집단의 평균이 가장 높다
>
> 오답을 고른 사람이 있다면, 왜 ①이 틀린지 옆 사람에게 설명해 보자. (30초)

정답은 **②**이다. ANOVA의 F-검정은 "적어도 하나의 쌍이 다르다"만 말해 준다. 어느 쌍인지, 그리고 모든 쌍이 다 다른지는 사후검정으로 확인해야 한다.

---

### 효과 크기와 가정 위반 대안

p값만으로는 차이의 **크기**를 알 수 없다. 따라서 효과 크기를 함께 보고한다.

| 지표 | 작은 효과 | 중간 효과 | 큰 효과 |
|---|---:|---:|---:|
| **Cohen's d** (t-검정) | 약 0.2 | 약 0.5 | 약 0.8+ |
| **η²** (ANOVA) | 약 0.01 | 약 0.06 | 약 0.14+ |

**표 W06.1** 가정 위반 시 대안 검정

| 가정 위반 | 대안 |
|---|---|
| 정규성 위반 | Mann-Whitney U (t 대안), Kruskal-Wallis (ANOVA 대안) |
| 등분산 위반 | Welch's t-검정 |
| 다중비교 | Tukey HSD, Bonferroni 보정 |

### 예시: 성별 × 소득 (t-검정)

social_survey_w06.csv 데이터에서 성별에 따른 소득 차이를 t-검정으로 분석한다.

```python
from scipy import stats
import pandas as pd
import numpy as np

df = pd.read_csv("social_survey_w06.csv")
male = df[df["성별"] == "남성"]["소득"]
female = df[df["성별"] == "여성"]["소득"]

# 가정 확인
stat_m, p_m = stats.shapiro(male)
stat_f, p_f = stats.shapiro(female)
lev_stat, lev_p = stats.levene(male, female)

# t-검정 실행
t_stat, p_val = stats.ttest_ind(male, female)
d = (male.mean() - female.mean()) / np.sqrt(((len(male)-1)*male.std()**2 + (len(female)-1)*female.std()**2) / (len(male)+len(female)-2))
```

_전체 코드는 practice/W06/code/W06-1-t-test-anova.py 참고_

**검정 결과:**

- Shapiro-Wilk: 남성 p > 0.05, 여성 p > 0.05 → 정규성 충족
- Levene 검정: p = 0.42 → 등분산 충족
- **t(198) = 2.45, p = 0.015**
- **Cohen's d = 0.35** (작은~중간 효과)
- 남성 평균 355만 원(SD 165), 여성 평균 310만 원(SD 148)

**결과 해석:** 유의수준 0.05에서, 남성과 여성의 평균 소득에는 통계적으로 유의한 차이가 있는 것으로 나타났다(t(198) = 2.45, p = 0.015). Cohen's d = 0.35로 작은~중간 수준의 효과이다. 단, 직종/경력/근무시간 등 교란변수를 통제하지 않았으므로 성별이 소득을 "결정한다"는 뜻은 아니다.

## 실습 과제 (수업 내 완결형)

### 기본(Basic) 트랙

#### ① 실행

성별(2집단) × 소득에 대해 독립표본 t-검정을 실행한다. **가정 확인을 반드시 먼저** 수행한다.

```python
from scipy import stats
import pandas as pd
import numpy as np

df = pd.read_csv("social_survey_w06.csv")
male = df[df["성별"] == "남성"]["소득"]
female = df[df["성별"] == "여성"]["소득"]

# 1. 정규성 확인 (Shapiro-Wilk)
print("=== 정규성 검정 ===")
print(f"남성: W = {stats.shapiro(male)[0]:.4f}, p = {stats.shapiro(male)[1]:.4f}")
print(f"여성: W = {stats.shapiro(female)[0]:.4f}, p = {stats.shapiro(female)[1]:.4f}")

# 2. 등분산 확인 (Levene)
lev_stat, lev_p = stats.levene(male, female)
print(f"\n=== 등분산 검정 (Levene) ===")
print(f"Levene: F = {lev_stat:.4f}, p = {lev_p:.4f}")

# 3. t-검정 실행
t_stat, p_val = stats.ttest_ind(male, female)
print(f"\n=== t-검정 결과 ===")
print(f"t({len(male)+len(female)-2}) = {t_stat:.3f}, p = {p_val:.4f}")
print(f"남성 평균: {male.mean():.1f}, 여성 평균: {female.mean():.1f}")
```

**완료 체크**: 정규성, 등분산, t-검정 결과가 모두 출력되었는가?

#### ② 확인

효과 크기를 계산하고, Copilot 응답을 확인하고 검증한다.

**2-1. Cohen's d 계산**:

```python
n1, n2 = len(male), len(female)
pooled_std = np.sqrt(((n1-1)*male.std()**2 + (n2-1)*female.std()**2) / (n1+n2-2))
d = (male.mean() - female.mean()) / pooled_std
print(f"Cohen's d = {d:.3f}")
```

**2-2. Copilot 프롬프트 (필수 1)**: Copilot에게 다음을 질문한다.

> "Levene 검정 결과 p=0.02이다. 등분산 가정이 충족되는가? 아니라면 어떻게 해야 하는가?"

Copilot의 답변을 읽고, **1가지를 채택 또는 거절**한다. 채택/거절 이유를 2문장 이내로 작성한다.

### 확장(Plus) 트랙

기본 트랙을 완료한 학생은 추가로 다음을 수행한다.

> 교육수준(3집단: 고졸/대졸/대학원) × 소득에 대해 ANOVA를 실행한다. 가정 확인(정규성, 등분산) → F-검정 → η² 계산 → **Tukey 사후검정**까지 수행한 뒤, 다음 질문에 답한다.
>
> "어느 집단 쌍에서 유의한 차이가 나타나는가? 사후검정 결과를 근거로 서술하라."

```python
from scipy import stats
from statsmodels.stats.multicomp import pairwise_tukeyhsd

high = df[df["교육수준"] == "고졸"]["소득"]
college = df[df["교육수준"] == "대졸"]["소득"]
grad = df[df["교육수준"] == "대학원"]["소득"]

# ANOVA
f_stat, p_val = stats.f_oneway(high, college, grad)
print(f"F(2, {len(df)-3}) = {f_stat:.2f}, p = {p_val:.4f}")

# Tukey 사후검정
tukey = pairwise_tukeyhsd(df["소득"], df["교육수준"], alpha=0.05)
print(tukey)
```

## 정리/오류 패턴 공유 + 비판적 읽기

### 흔한 오류 3개

실습 중 자주 나타나는 오류를 정리한다. 자신의 결과에 해당하는 항목이 없는지 점검해 보자.

**오류 1: 등분산 가정 미확인** -- `stats.ttest_ind()`의 기본 설정은 등분산을 가정한다(`equal_var=True`). Levene 검정 없이 실행하면 등분산 위반 시 잘못된 결과를 얻는다. Levene p < 0.05이면 `equal_var=False`로 Welch's t를 사용한다.

**오류 2: ANOVA 유의 → "모든 집단이 다르다"** -- F-검정은 "적어도 하나의 쌍이 다르다"만 알려준다. 어느 쌍인지는 사후검정으로 확인해야 한다.

**오류 3: 표본이 작은데 정규성 무시** -- n < 30이면 Shapiro-Wilk 검정으로 정규성을 확인한다. 위반 시 Mann-Whitney U 또는 Kruskal-Wallis를 사용한다.

---

### 해석문 모범 구조

오늘 과제의 (D)항 해석문은 다음 4단계 구조를 따른다.

> **(분석목적)** 성별에 따라 평균 소득에 차이가 있는지 확인하기 위해 독립표본 t-검정을 실시했다.
>
> **(핵심근거)** 정규성(Shapiro-Wilk: 남성 p > 0.05, 여성 p > 0.05)과 등분산(Levene p = 0.42) 가정이 모두 충족되었다. t(198) = 2.45, p = 0.015이며, Cohen's d = 0.35이다. 남성 평균 소득은 355만 원(SD 165), 여성 평균 소득은 310만 원(SD 148)이다.
>
> **(결론)** 유의수준 0.05에서, 성별에 따른 평균 소득 차이는 통계적으로 유의하다. Cohen's d = 0.35로 작은~중간 수준의 효과이다.
>
> **(주의)** 이 결과는 성별이 소득을 "결정한다"는 의미가 아니다. 직종, 경력, 근무시간, 교육수준 등 교란변수를 통제하지 않았으며, 가상 데이터(n = 200)의 표본 한계가 있다.

### 비판적 읽기 (5–10분)

아래는 가상의 뉴스 기사 발췌문이다. 읽고 문제점을 찾아보자.

> **〈○○일보〉 2026.03.10**
>
> *"국내 직장인 800명을 대상으로 조사한 결과, 대학원 졸업자의 평균 연봉이 고졸 이하보다 1,200만 원 높은 것으로 나타나, **교육이 소득을 높이는 핵심 요인으로 확인되었다**. 연구진은 학력이 높을수록 소득이 올라간다고 결론지었다."*

> **토론 질문**: 이 기사의 해석에서 문제점을 찾아보자. (2–3분, 짝 토론 후 전체 공유)

**문제점 예시:**

1. **인과 표현**: "교육이 소득을 **높이는** 핵심 요인"이라고 서술했다. 횡단면 조사로는 연관만 확인할 수 있으며, 인과를 주장하려면 실험이나 종단 연구가 필요하다. "교육수준에 따라 소득에 차이가 **관찰되었다**"가 적절하다.

2. **효과 크기 미보고**: p값이나 효과 크기(Cohen's d, η²)가 전혀 보고되지 않았다. 800명이라는 큰 표본에서는 사소한 차이도 유의하게 나올 수 있으므로, 효과 크기 없이는 실질적 의미를 판단할 수 없다.

3. **교란변수 무시**: 연령, 직종, 근무시간, 부모 소득 등 교육수준과 소득 모두에 관련될 수 있는 변수가 통제되지 않았다. "학력이 높을수록 소득이 올라간다"는 단순화된 결론은 과잉 해석이다.

### Exit Ticket 문항

> **Q.** ANOVA 결과 F(2, 197) = 5.23, p = 0.006, η² = 0.05가 나왔다. 이 결과만 가지고 "세 집단의 평균이 모두 서로 다르다"고 결론 내릴 수 있는가? 이유를 1–2문장으로 서술하라.

**모범 답안 예시**: "결론 내릴 수 없다. ANOVA의 F-검정은 '적어도 하나의 집단 쌍에서 유의한 차이가 있다'만 알려줄 뿐, 어느 쌍이 다른지는 사후검정(Tukey HSD 등)으로 확인해야 한다."
