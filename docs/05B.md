# W05B. 가설--분석 선택--결론 문장화 실습 — 과제 5

> 연구 질문 → 가설 → 분석 선택 → 결론의 논리적 흐름을 문장으로 쓸 수 있는가?

**학습 목표**
1. 연구 질문에 맞는 **귀무가설/대립가설**을 설정하고, 검정 선택 가이드표를 참조하여 **적절한 검정**을 고를 수 있다.
2. Copilot(바이브코딩)을 활용하여 검정 선택의 타당성을 **확인**하고, 채택/거절 근거를 작성할 수 있다.
3. 가상 결과를 기반으로 **A-E 보고서**를 작성하여 클래스룸에 제출할 수 있다.

## (10-15분) 과제 자료 안내 + 바이브코딩 가이드

### 오늘의 과제 특성

오늘 과제는 이전 주차와 다르게, **실제 데이터 분석 코드를 실행하지 않는다**. 대신 "연구 질문 → 가설 → 검정 선택 → 가상 결과 해석"이라는 **논리적 흐름을 문장으로 쓰는 연습**에 집중한다. 코드 실행은 6주부터 본격적으로 시작한다.

### 검정 선택 가이드표 (참조용)

| 연구 질문 | 독립변수 척도 | 종속변수 척도 | 검정 |
|---|---|---|---|
| 범주 간 관계 | 명목 | 명목 | 카이자승 |
| 두 집단 평균 차이 | 명목(2수준) | 등간/비율 | 독립표본 t-검정 |
| 세 집단 이상 평균 차이 | 명목(3수준+) | 등간/비율 | ANOVA |
| 두 연속 변수 관계 | 등간/비율 | 등간/비율 | 상관/회귀 |

### 바이브코딩 활용

오늘은 코드 생성 대신, Copilot에게 **검정 선택의 적절성을 확인받는 용도**로 바이브코딩을 활용한다.

**바이브코딩 프롬프트 예시**:

| 순서 | 프롬프트 예시 |
|---:|---|
| 1 | "내 연구 질문이 '성별에 따라 월 소득에 차이가 있는가?'이고, 독립변수는 성별(명목, 2수준), 종속변수는 월 소득(비율)이다. 어떤 검정을 써야 하는지 알려줘." |
| 2 | "교육수준이 3수준(고졸/대졸/대학원)이고 종속변수가 생활만족도(5점 척도)일 때, t-검정 대신 ANOVA를 써야 하는 이유를 설명해줘." |
| 3 | "t(198) = 2.45, p = 0.015, Cohen's d = 0.35일 때, 이 결과를 인과가 아닌 표현으로 해석하는 문장을 써줘." |

**주의사항**:
- Copilot의 검정 선택 제안을 **검정 선택 가이드표와 대조**하여 확인한다.
- Copilot이 제시한 해석문은 참고만 하고, 해석문(D항)은 직접 작성한다.
- Copilot 사용 내역은 반드시 (E)항에 기록한다.

### 과제 1: 가설 설정 + 검정 선택 (12분)

아래 3개의 연구 질문에 대해 각각 **귀무가설(H₀)**, **대립가설(H₁)**, **적절한 검정**을 작성한다.

**연구 질문 1**: "성별에 따라 월 소득에 차이가 있는가?"
- 독립변수: 성별 (명목, 2수준: 남성/여성)
- 종속변수: 월 소득 (비율)

**연구 질문 2**: "교육수준에 따라 생활만족도가 다른가?"

- 종속변수: 생활만족도 (서열/등간, 5점 척도)

**연구 질문 3**: "연령과 소득 사이에 관계가 있는가?"
- 독립변수: 연령 (비율)

### 과제 3: A-E 보고서 작성 (16분)

두 조원이 함께 3개 연구 질문 중 **1개를 골라**, 아래 가상 결과를 기반으로 A-E 보고서를 작성한다.

#### 가상 결과 (택 1)

**질문 1 가상 결과**:
- 검정: 독립표본 t-검정
- t(198) = 2.45, p = 0.015, Cohen's d = 0.35
- 남성 평균: 355만 원, 여성 평균: 310만 원

**질문 2 가상 결과**:
- 검정: ANOVA
- F(2, 197) = 4.82, p = 0.009, eta^2 = 0.047
- 고졸 평균: 2.85, 대졸 평균: 3.15, 대학원 평균: 3.52

**질문 3 가상 결과**:
- 검정: 상관/단순회귀
- r = 0.42, p < 0.001
- beta_1 = 5.2 (연령 1세 증가 시 소득 5.2만 원 증가 추정)

#### A-E 보고서 작성

- **(A) 분석 질문** (1-2문장): "___와(과) ___ 사이에 통계적으로 유의한 차이/관계가 있는가?"
- **(B) 사용 변수/척도**: 변수명(척도), 각 변수의 수준 나열
- **(C) 결과 근거**: 검정통계량, df, p값, 효과 크기
- **(D) 해석문** (5-7문장): 아래 4단계 구조를 따른다.
  - **(분석목적)** 왜 이 분석을 했는가?
  - **(핵심근거)** 검정통계량, df, p값, 효과 크기는 얼마인가?
  - **(결론)** 귀무가설을 기각하는가? 효과 크기의 수준은?
  - **(주의)** 이 결과는 인과가 아니다. 어떤 한계가 있는가?
- **(E) Copilot 사용 요약** (2-3줄): 어떤 프롬프트를 사용했고, 채택/거절 판단은 무엇이었는가?

### 해석문(D항) 작성 가이드

조원 두 명이 초안을 함께 읽고, **"인과 표현"이 있는지 반드시 교차 점검**한다.

| 잘못된 표현 (X) | 올바른 표현 (O) |
|---|---|
| "성별이 소득에 영향을 미친다" | "성별에 따라 소득에 통계적으로 유의한 차이가 관찰된다" |
| "교육수준이 높을수록 만족도가 올라간다" | "교육수준이 높은 집단에서 만족도 평균이 더 높게 나타난다" |
| "연령이 소득을 결정한다" | "연령과 소득 사이에 양의 상관이 관찰된다" |

**효과 크기 해석 기준** (참고):

| 지표 | 작은 효과 | 중간 효과 | 큰 효과 |
|---|---|---|---|
| Cohen's d | 0.2 | 0.5 | 0.8 |
| eta^2 | 0.01 | 0.06 | 0.14 |
| r | 0.1 | 0.3 | 0.5 |

## (60-80분) 조별 발표 + 전체 토론

### 조별 발표

2-3개 조가 자신의 결과를 **1분 내외**로 발표한다. 발표 형식은 다음과 같다.

> "우리 조는 질문 ___을(를) 선택했다. H₀는 '___'이고, ___검정을 선택했다. 가상 결과에서 검정통계량 = ___, p = ___, 효과 크기 = ___이었다. 따라서 유의수준 0.05에서 H₀를 ___(기각/기각 못함)하며, 효과 크기는 ___ 수준이다."

### 전체 토론

발표 후 아래 질문에 대해 전체 토론한다.

> **질문**: "3개 연구 질문 중 어느 것이 가설 설정이 가장 어려웠는가? 그 이유는?"
