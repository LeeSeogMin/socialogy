# W14. 내 분석을 5분 안에 설득력 있게 전달할 수 있는가? — 최종 발표 전반부

> 4-5명 세션에서 개인 발표를 수행하고, 세션 내 동료 피드백을 주고받을 수 있는가?

**학습 목표**
1. 13주 교차리뷰와 발표 리허설 결과를 반영하여 **발표 자료를 최종 완성**할 수 있다.
2. 5분 발표와 2분 Q&A를 통해 본인 분석의 **핵심 메시지("So what?")** 를 전달할 수 있다.
3. 세션 내 다른 발표자에게 **구체적 피드백**을 작성할 수 있다.

---

## A회차: 최종 발표 준비와 세션 운영

### 세션 구성

전체 수강생을 **4-5명 단위의 소세션**으로 나눈다. 14주에 절반, 15주에 나머지 절반이 발표한다.

| 단위 | 시간 | 내용 |
|---:|---|---|
| 개인 | 5분 | 발표 |
| 개인 | 2분 | 동료 + 교수 Q&A |
| 세션 말 | 5분 | 교수 코멘트 |

---

### 발표 시 반드시 지킬 3가지

1. **인과 표현 금지**: "A가 B를 일으킨다" (X) → "A와 B 사이에 통계적으로 유의한 관련이 있다" (O)

2. **한계를 솔직히**: 데이터의 제약, 표본의 한계, 분석 기법의 제한을 숨기지 않는다. 한계를 인정하는 것은 약점이 아니라 학술적 성실성이다.

3. **"모르겠다"의 용기**: Q&A에서 모르는 질문에 "잘 모르겠지만, ___를 추가로 분석하면 답할 수 있을 것이다"로 대응한다. 근거 없는 추측은 하지 않는다.

---

### Q&A 예상 질문 대비

교수와 동료가 흔히 하는 질문 유형:

| 질문 | 준비할 답변 방향 |
|---|---|
| 왜 이 분석 기법을 선택했는가? | 변수 척도 + 연구 질문의 유형으로 설명 |
| 표본 크기가 충분한가? | N과 검정력 고려, 일반화 가능 범위 |
| 이 결과를 일반화할 수 있는가? | 표본 특성 + 조사 시점 + 지역 한계 명시 |
| 다른 변수를 통제하면 달라질 수 있지 않은가? | 통제한 변수 + 누락된 변수 솔직 인정 |
| 이분형인데 왜 OLS인가? (또는 역) | 종속변수 분포 근거로 설명 |

## B회차: 쇼케이스 전반부 — 개인 발표

### 발표 진행

- 각 세션은 **4-5명**으로 구성한다.
- 세션별 사회자는 교수 또는 지정된 학생이 맡는다.
- 발표 순서는 사전 배정(또는 추첨).

---

### 최종 제출물 (14주 또는 15주 발표일까지)

| 제출물 | 형식 | 비고 |
|---|---|---|
| **완성 리서치 브리프** | .pdf 또는 .docx | 6-8쪽, 7개 섹션 포함 |
| **재현 가능 노트북** | .ipynb | "Restart & Run All" 완주 필수 |
| **발표 슬라이드** | .pdf 또는 .pptx | 5-7장 |
| **세션 피어 피드백** | Google Classroom 양식 | 본인 세션 전원에 대해 |

---

### 평가 기준 (개인 과제 100% 기준)

| 평가 항목 | 비율 | 세부 |
|---|---:|---|
| **분석 타당성** | 25% | 척도·가정에 맞는 기법 선택, 가정 확인, 효과크기 보고 |
| **해석 정확성** | 25% | 비인과 표현, 한계 명시, 선행연구 비교 |
| **재현성** | 15% | 노트북 재실행 성공, 데이터 경로·시드 설정 |
| **발표 전달력** | 15% | 5분 엄수, 표/그림 1개 이상 활용, "So what" 명확 |
| **Q&A 대응** | 10% | "모르겠다"의 솔직함 + 근거 기반 답변 |
| **세션 기여** | 10% | 다른 발표자에 대한 구체 피드백 작성 |

**주의**: 분석 타당성과 해석 정확성은 **주제 독립적 기준**으로 평가한다. 주제의 참신성 자체는 가산점이 아니다.
