# W07B. 단순회귀 실습 — 과제 7

> 주어진 데이터로 단순회귀를 실행하고, 계수·R²·잔차를 해석하며, "인과 아님"을 명시한 보고서를 작성할 수 있는가?

**학습 목표**
1. Copilot(바이브코딩)을 활용하여 산점도, 단순회귀, R² 확인, 잔차 플롯을 **수행**할 수 있다.
2. 회귀 계수(β₁)를 구체적인 문장으로 해석하되, **인과가 아닌 예측/연관**으로 서술할 수 있다.
3. 두 회귀 모형의 결과를 비교하고, 코드와 함께 **A-E 보고서**를 작성하여 클래스룸에 제출할 수 있다.

### 바이브코딩이란?

오늘 실습에서는 **바이브코딩(vibe coding)** 방식으로 분석을 수행한다. 바이브코딩이란 Copilot에게 데이터와 분석 목표를 자연어로 설명하면, Copilot이 코드를 생성해 주는 방식이다. 학생은 생성된 코드를 실행하고, 결과를 확인하며, 해석문을 작성한다.

**바이브코딩 프롬프트 예시**:

| 순서 | 프롬프트 예시 |
|---:|---|
| 1 | "social_survey_w07.csv를 불러와서 교육연수와 소득의 산점도를 그리고, 단순회귀(OLS)를 실행해줘. summary()를 출력해줘." |
| 2 | "위 모형의 잔차 플롯(예측값 vs 잔차)을 그려줘." |
| 3 | "연령과 생활만족도로 같은 분석(산점도 + 단순회귀 + 잔차 플롯)을 해줘." |
| 4 | "두 모형의 β₁, p값, R²를 비교하는 표를 만들어줘." |

**주의사항**:
- Copilot이 생성한 코드를 그대로 실행하기 전에, 변수명이 데이터와 일치하는지 확인한다.
- **최종 코드와 실행 결과를 모두 과제물에 포함**한다.
- Copilot의 해석 문장은 참고만 하고, 해석문은 직접 작성한다.
- 해석문에 **"인과 아님"을 반드시 명시**한다.

#### ② A-E 보고서 작성

과제 1과 과제 2의 결과를 **모두 포함**하여 아래 형식으로 작성한다.

- **(A) 분석 질문** (1-2문장): "교육 연수는 소득을, 연령은 생활만족도를 통계적으로 유의하게 예측하는가?"
- **(B) 사용 변수/척도**: 변수명(비율/등간), 각 변수의 단위/범위 나열

- **(E) Copilot 사용 요약** (2-3줄): 어떤 프롬프트를 사용했고, 코드 생성 과정에서 수정한 부분이 있었는가?

### 해석문(D항) 작성 가이드

| 잘못된 표현 (X) | 올바른 표현 (O) |
|---|---|
| "교육이 소득에 영향을 미친다" | "교육 연수와 소득 사이에 유의한 양의 연관이 관찰된다" |
| "교육을 받으면 소득이 높아진다" | "교육 연수가 높을수록 소득이 높은 경향이 추정된다" |
| "연령 때문에 만족도가 낮아진다" | "연령과 생활만족도 사이에 통계적으로 유의한 연관이 관찰되지 않았다" |

**해석문에 반드시 포함할 문장**: "이 결과는 ___를 의미하지 않으며(인과 아님), ___의 한계가 있다."

### 확장(Plus) 트랙

기본 과제를 완료한 조는 추가로 다음을 수행한다.

> **소득 → 생활만족도** 단순회귀를 추가로 실행한다. 3개 모형(교육연수→소득, 연령→생활만족도, 소득→생활만족도)의 R²를 비교하는 표를 작성하고, "어떤 예측 관계가 가장 강한가?"를 1문장으로 답한다.
