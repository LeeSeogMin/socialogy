### 분석 참고 코드

아래 코드를 기본 틀로 사용한다. 변수명만 바꿔서 적용하면 된다.

```python
import pandas as pd
from scipy.stats import chi2_contingency
import numpy as np

df = pd.read_csv("social_survey_w04.csv")

# 분할표 생성
ct = pd.crosstab(df["변수1"], df["변수2"], margins=True)
print(ct)

# 카이자승 검정
ct_core = pd.crosstab(df["변수1"], df["변수2"])
chi2, p, dof, expected = chi2_contingency(ct_core)
print(f"\nχ²({dof}) = {chi2:.3f}, p = {p:.4f}")

# 기대빈도 확인
print("\n=== 기대빈도 ===")
print(pd.DataFrame(expected, index=ct_core.index,
                    columns=ct_core.columns).round(2))
print(f"최소 기대빈도: {expected.min():.2f}")

# Cramér's V
n = ct_core.sum().sum()
V = np.sqrt(chi2 / (n * (min(ct_core.shape) - 1)))
print(f"Cramér's V = {V:.3f}")
```

#### ③ 종합 해석문 작성 (10분)

조원 전원이 합의하여 **과제 1 또는 과제 2 중 1개**를 골라 A–E 형식의 해석문을 완성한다.

- **(B) 사용 변수/척도**: 변수명(명목/서열), 각 변수의 수준 나열
- **(C) 결과 근거**: 분할표 + χ²(df), p값 + 기대빈도 확인 결과 + Cramér's V
- **(D) 해석문** (5–7문장): 아래 4단계 구조를 따른다.

  - **(주의)** 이 결과는 인과가 아니다. 어떤 한계가 있는가?
- **(E) Copilot 사용 요약** (2–3줄): 어떤 질문을 했고, 어떤 답변을 받았으며, 무엇을 채택/거절했는가?

**해석문(D항) 작성 시 주의**: 조원 전원이 초안을 읽고, **"인과 표현"이 있는지 반드시 교차 점검**한다.

| 잘못된 표현 (X) | 올바른 표현 (O) |
|---|---|

| "연령 때문에 SNS 이용이 줄어든다" | "연령대에 따라 SNS 이용 비율에 차이가 나타난다" |
| "남성이라서 A당을 지지한다" | "남성에서 A당 지지 비율이 상대적으로 높게 나타난다" |

|---|---|---|---|---|---|

### 흔한 오류 점검 (3분)

교수가 순회하며 발견한 오류 패턴을 공유한다.

**오류 1: 기대빈도 확인 누락**
- `chi2_contingency()`의 4번째 반환값이 기대빈도. 반드시 `print(expected)`로 확인한다.

**오류 2: Cramér's V 미보고**
- p값만 보고하면 효과 크기를 판단할 수 없다. V < 0.1(약함), 0.1–0.3(중간), > 0.3(강함).

**오류 3: 인과 표현 사용**
- "영향을 미친다", "결정한다", "~때문에" → 모두 비인과 표현으로 수정.

### 해석문 모범 구조 (교수용)

> **(분석목적)** 성별과 지지정당 사이에 통계적으로 유의한 연관이 있는지 확인하기 위해 카이자승 검정을 실시했다.
>
> **(핵심근거)** 분할표에서 남성은 A당(43.3%), 여성은 B당(28.1%)과 C당(30.2%)을 상대적으로 더 지지하는 경향이 나타났다. χ²(3) = 16.262, p = 0.001이며, Cramér's V = 0.285이다. 기대빈도는 모든 셀에서 5 이상(최솟값 20.64)으로 검정 가정을 충족했다.
>
> **(결론)** 유의수준 0.05에서 귀무가설(성별과 지지정당은 독립이다)을 기각한다. 두 변수 사이에 중간 수준의 연관이 관찰된다.
>
> **(주의)** 이 결과는 성별이 정당 선택을 "결정"하거나 "원인"이라는 의미가 아니다. 연령, 소득, 지역 등 교란변수를 고려하지 않았으며, 설문 표본(n=200)의 대표성에도 한계가 있다.
