### 오늘의 질문

A회차(W04A)에서는 분할표, 카이자승 검정, 기대빈도, Cramér's V의 개념을 배웠다. 오늘은 이 도구들을 **실제 데이터에 직접 적용**하는 시간이다. 교수가 제시한 예시를 따라하는 것이 아니라, 학생 스스로 변수를 선택하고, 검정을 실행하며, 결과를 해석문으로 완성해야 한다.

핵심 질문은 하나이다: **"실제 데이터로 두 범주형 변수의 관계를 검정하고 해석할 수 있는가?"**

### 빠른 진단 (1문항)

> **Q.** 카이자승 검정의 가정 2가지를 적어라. 기대빈도가 5 미만인 셀이 있으면 어떻게 하는가?

**모범 답안**:
- 가정 1: **독립 표본** -- 각 관찰(응답자)이 서로 독립적이어야 한다.
- 가정 2: **기대빈도 >= 5** -- 모든 셀의 기대빈도가 5 이상이어야 한다.
- 기대빈도가 5 미만인 셀이 있으면: **Fisher 정확 검정(Fisher's exact test)**으로 전환하거나, 빈도가 적은 범주를 통합하여 재분석한다.

*(1분간 적고, 2--3명에게 구두로 답변을 받은 뒤, 간략히 확인한다.)*

## (10--30분) 미니 강의 (복습 + 실습 안내)

### W04A 핵심 복습 (5분)

B회차이므로 새로운 개념을 도입하지 않는다. A회차에서 배운 전체 흐름을 한 장으로 정리한다.

**카이자승 검정 흐름 요약:**

```
분할표 생성 → 기대빈도 확인(모두 >= 5?) → χ² 계산 → p값 확인 → Cramér's V → 해석문 작성
```

각 단계에서 확인해야 할 것을 다시 짚는다.

| 단계 | 확인 사항 |
|---|---|
| 분할표 | 관찰빈도가 올바르게 집계되었는가? 행/열 합계가 전체 n과 일치하는가? |
| 기대빈도 | 모든 셀이 5 이상인가? 5 미만이면 Fisher 정확 검정 또는 범주 통합 고려 |
| χ² / p값 | χ²(df) = ___, p = ___. p < 0.05이면 귀무가설(독립) 기각 |
| Cramér's V | V < 0.1 약한, 0.1--0.3 중간, > 0.3 강한 연관 |
| 해석문 | 비인과 표현 사용, 한계 명시 |

### 사용 가능한 변수 쌍 안내 (5분)

아래는 분석 가능한 변수 쌍의 예시이다. 학생은 이 중 **1개 이상을 자유롭게 선택**하여 분석한다.

| 변수 쌍 | 예상 질문 |
|---|---|
| 성별 x 지지정당 | 성별에 따라 지지정당의 분포가 다른가? |

| 교육수준 x SNS이용여부 | 교육수준에 따라 SNS 이용 비율이 다른가? |

**권장**: 옆 사람과 **다른 조합**을 선택하도록 한다. 정리 시간에 서로 다른 결과를 비교하기 위함이다.

### 실습 코드 참조

오늘 실습에 필요한 코드의 기본 구조는 다음과 같다.

```python
import pandas as pd
from scipy.stats import chi2_contingency
import numpy as np

# 1. 데이터 불러오기
df = pd.read_csv("social_survey_w04.csv")

# 2. 분할표 생성
ct = pd.crosstab(df["변수1"], df["변수2"], margins=True)
print("=== 분할표 (관찰빈도) ===")
print(ct)

# 3. 카이자승 검정 실행
ct_no_margins = pd.crosstab(df["변수1"], df["변수2"])
chi2, p, dof, expected = chi2_contingency(ct_no_margins)
print(f"\nχ²({dof}) = {chi2:.3f}, p = {p:.4f}")

# 4. 기대빈도 확인
print("\n=== 기대빈도 ===")
print(pd.DataFrame(expected, index=ct_no_margins.index,
                    columns=ct_no_margins.columns).round(2))
print(f"최소 기대빈도: {expected.min():.2f}")

# 5. Cramér's V 계산
n = ct_no_margins.sum().sum()
V = np.sqrt(chi2 / (n * (min(ct_no_margins.shape) - 1)))
print(f"\nCramér's V = {V:.3f}")
```

### 확장(Plus) 트랙

기본 트랙을 완료한 학생은 추가로 다음을 수행한다.

**3개 이상의 범주형 변수 조합을 비교 분석한다.**

아래 3가지 분석 모두를 실행하고, Cramér's V를 비교하여 "어떤 조합이 가장 강한 연관을 보이는가?"를 서술한다.

**분석 1: 성별 x 지지정당**

**표 W04.3** 성별 x 지지정당 분할표 (관찰빈도)

| | A당 | B당 | C당 | 무당파 | 합계 |

| 여성 | 18 | 27 | 29 | 22 | 96 |
| 합계 | 63 | 43 | 48 | 46 | 200 |

- 기대빈도: 모두 >= 5 (최솟값 20.64)

**표 W04.4** 연령대 x SNS이용여부 분할표 (관찰빈도)

| 30대 | 21 | 31 | 52 |
| 40대 | 23 | 29 | 52 |
| 50대+ | 27 | 13 | 40 |
| 합계 | 80 | 120 | 200 |

- chi2(3) = 26.355, p < 0.0001
- Cramér's V = 0.363
- 기대빈도: 모두 >= 5 (최솟값 16.00)

**분석 3: 성별 x 교육수준**

**표 W04.5** 성별 x 교육수준 분할표 (관찰빈도)

### 흔한 오류 3개

실습 중 자주 나타나는 오류를 정리한다. 자신의 과제에 해당하는 항목이 없는지 점검해 보자.

**오류 1: 기대빈도 확인을 건너뜀**

카이자승 검정을 실행한 뒤 p값만 확인하고, 기대빈도를 출력하지 않는 경우가 많다. `chi2_contingency()`는 4개의 값을 반환하며, 네 번째 값이 기대빈도 배열(`expected`)이다. 반드시 다음을 실행하여 확인한다.

```python
chi2, p, dof, expected = chi2_contingency(ct)
print(expected)  # 모든 값이 5 이상인지 눈으로 확인
```

기대빈도가 5 미만인 셀이 있으면, Fisher 정확 검정(`scipy.stats.fisher_exact`, 2x2 표의 경우)을 사용하거나 빈도가 적은 범주를 통합하여 재분석해야 한다.

**오류 2: Cramér's V를 보고하지 않음**

p값이 유의하다는 것만 보고하고 효과 크기를 생략하는 경우이다. "유의하지만 효과가 작을 수 있다" -- V값 없이는 연관의 실질적 의미를 판단할 수 없다. 반드시 다음을 함께 보고한다.

```python
n = ct.sum().sum()
V = np.sqrt(chi2 / (n * (min(ct.shape) - 1)))
print(f"Cramér's V = {V:.3f}")
```

| Cramér's V | 연관의 강도 |
|---:|---|
| < 0.1 | 약한 연관 (실질적 의미 작음) |
| 0.1 -- 0.3 | 중간 수준의 연관 |
| > 0.3 | 강한 연관 |

**오류 3: 해석문에 인과 표현 사용**

가장 흔하고 가장 심각한 오류이다. 카이자승 검정은 두 변수 간 **통계적 연관의 유무**만 판단한다. 인과의 방향이나 메커니즘에 대해서는 아무것도 말하지 않는다.

| 잘못된 표현 (X) | 올바른 표현 (O) |
|---|---|
| "성별이 정당 선택에 영향을 미친다" | "성별과 지지정당 사이에 통계적으로 유의한 연관이 관찰된다" |
| "연령 때문에 SNS 이용이 줄어든다" | "연령대에 따라 SNS 이용 비율에 차이가 나타난다" |
| "남성이라서 A당을 지지한다" | "남성에서 A당 지지 비율이 상대적으로 높게 나타난다" |

### 해석문 모범 답안 (교수용 -- 성별 x 지지정당 기준)

학생들의 해석문과 비교하기 위한 모범 구조이다. 정리 시간에 화면에 띄워 전체가 함께 확인한다.

> **(분석목적)** 성별과 지지정당 사이에 통계적으로 유의한 연관이 있는지 확인하기 위해 카이자승 검정을 실시했다.
>
> **(핵심근거)** 분할표에서 남성은 A당(43.3%), 여성은 B당(28.1%)과 C당(30.2%)을 상대적으로 더 지지하는 경향이 나타났다. chi2(3) = 16.262, p = 0.001이며, Cramér's V = 0.285이다. 기대빈도는 모든 셀에서 5 이상(최솟값 20.64)으로 검정 가정을 충족했다.
>
> **(결론)** 유의수준 0.05에서 귀무가설(성별과 지지정당은 독립이다)을 기각한다. 두 변수 사이에 중간 수준의 연관이 관찰된다.
>
> **(주의)** 이 결과는 성별이 정당 선택을 "결정"하거나 "원인"이라는 의미가 아니다. 연령, 소득, 지역 등 교란변수를 고려하지 않았으며, 설문 표본(n=200)의 대표성에도 한계가 있다.

### 비판적 읽기 (5--10분)

A회차에서는 가상의 뉴스 기사를 통해 인과 표현, 효과 크기 미보고, 교란변수 무시의 문제를 짚었다. 오늘은 학생들의 **실제 해석문**을 대상으로 같은 점검을 실시한다.

**절차:**

1. 교수가 실습 중 순회하며 발견한 해석문 2--3개를 익명으로 화면에 띄운다. (또는 자원자를 받는다.)
2. 전체 학생이 다음 3가지를 점검한다.
   - **인과 표현**: "영향을 미친다", "결정한다", "~때문에" 등이 있는가?
   - **효과 크기 누락**: Cramér's V가 보고되지 않았는가?
   - **한계 미언급**: 교란변수, 표본 한계 등이 빠져 있는가?
3. 문제가 발견되면 전체 토론으로 수정 방향을 함께 논의한다.

> **점검 예시**:
>
> (학생 해석문 발췌) "성별이 정당 선택에 영향을 미치는 것으로 나타났다. chi2(3) = 16.262, p = 0.001이다."
>
> 문제점:
> - "영향을 미치는" --> 인과 표현 (X). "통계적으로 유의한 연관이 관찰된다" (O)
> - Cramér's V 미보고 --> V = 0.285로 중간 수준의 연관임을 명시해야 한다
> - 한계 미언급 --> 교란변수(연령, 소득 등)와 표본 대표성 한계를 추가해야 한다

## (85--90분) Exit Ticket

### Exit Ticket 문항

> **Q.** 오늘 분석한 변수 쌍의 결과를 1문장으로 요약해라. (반드시 chi2값, p값, Cramér's V, "연관" 표현을 포함할 것)

**작성 형식**: "___와(과) ___ 사이에 [통계적으로 유의한/유의하지 않은] 연관이 [관찰되었다/관찰되지 않았다](chi2(df) = ___, p = ___, Cramér's V = ___)."

**모범 답안 예시**:
- "성별과 지지정당 사이에 통계적으로 유의한 중간 수준의 연관이 관찰되었다(chi2(3) = 16.262, p = 0.001, Cramér's V = 0.285)."
- "연령대와 SNS이용여부 사이에 통계적으로 유의한 강한 연관이 관찰되었다(chi2(3) = 26.355, p < 0.0001, Cramér's V = 0.363)."
- "성별과 교육수준 사이에 통계적으로 유의한 연관은 관찰되지 않았다(chi2(2) = 3.221, p = 0.200, Cramér's V = 0.127)."

### 다음 시간 예고

3--4주에서는 "이 데이터에서 무엇이 보이는가"를 분석했다(기술통계 + 카이자승). 다음 주(5주)에는 **"이 패턴이 모집단에서도 성립하는가?"**라는 질문으로 넘어간다 -- 기술통계에서 **추론통계(inferential statistics)**로의 전환이다.
